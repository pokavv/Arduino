# PCF8574 I2C GPIO 확장 IC

## 개요

PCF8574는 NXP Semiconductors(구 Philips)가 개발한 I2C 인터페이스 기반 8비트 GPIO 확장 IC입니다. I2C 버스(SDA/SCL) 2개 핀만으로 8개의 준양방향(Quasi-bidirectional) I/O 포트를 추가할 수 있습니다. 주소 핀(A0~A2) 3개로 최대 8개까지 같은 버스에 연결 가능(총 64개 GPIO 확장).

**PCF8574 vs PCF8574A:**
- PCF8574:  I2C 주소 0x20 ~ 0x27
- PCF8574A: I2C 주소 0x38 ~ 0x3F

**주요 용도**: LCD I2C 어댑터 내장 IC, LED/버튼 확장, 키패드 인터페이스, I2C 기반 GPIO 확장.

## 동작 원리

PCF8574의 I/O 포트는 **준양방향(Quasi-bidirectional)** 방식입니다.

**출력 모드:**
- I2C로 포트 레지스터에 '0' 쓰기 → 해당 핀이 LOW (수십 mA sink 가능)
- I2C로 포트 레지스터에 '1' 쓰기 → 해당 핀이 내부 약한 풀업(300µA)으로 HIGH

**입력 모드:**
- 먼저 해당 핀에 '1'을 쓰기 (내부 풀업 활성화)
- 외부 장치가 핀을 LOW로 당기면 → I2C 읽기 시 '0'으로 읽힘
- 외부 아무것도 없으면 내부 풀업에 의해 '1'로 읽힘

**인터럽트(INT 핀):**
- I/O 포트 상태가 변하면 자동으로 INT 핀이 LOW가 됨
- 아두이노 인터럽트 핀에 연결하면 폴링 없이 입력 변화 감지 가능
- I2C 읽기 후 INT 핀 자동 복귀

### 출력 동작

| I2C 전송 데이터 (Px) | 핀 상태 | 출력 전류 |
|---------------------|---------|---------|
| 0 | LOW | 최대 25 mA (싱크) |

### 입력 동작

| 핀에 '1' 쓴 후 외부 신호 | I2C 읽기 결과 |
|------------------------|-------------|
| 외부 오픈 (아무것도 연결 안 됨) | 1 |
| 외부 GND에 연결 (또는 스위치 누름) | 0 |
| 외부 HIGH 신호 | 1 |

### 전체 포트 동작 예시

| I2C Write 값 | P7~P0 상태 (이진) | LED 상태 (Low Active 기준) |
|-------------|-----------------|--------------------------|
| 0x00 | 00000000 | 전체 ON |
| 0xFF | 11111111 | 전체 OFF |
| 0x0F | 00001111 | P4~P7 ON, P0~P3 OFF |

## 아두이노 연결 방법

### 기본 연결

```
ESP32-C3 Super Mini       PCF8574
───────────────────       ────────
3.3V ─────────────────→ VCC (16)
GND  ─────────────────→ GND (8)
G6 (SDA) ─────────────→ SDA (15)  ← 4.7kΩ 풀업 저항 (SDA-VCC)
G7 (SCL) ─────────────→ SCL (14)  ← 4.7kΩ 풀업 저항 (SCL-VCC)
GND  ─────────────────→ A0 (1)   ← 주소: 0x20
GND  ─────────────────→ A1 (2)
GND  ─────────────────→ A2 (3)
(옵션) G3 ────────────→ INT (13) ← 10kΩ 풀업 저항 (INT-VCC)
```

> ESP32-C3 기본 I2C: SDA=G6, SCL=G7 (보드에 따라 상이, `Wire.begin(SDA, SCL)`)

### I2C 풀업 저항

I2C 라인에 풀업 저항이 없으면 통신 불가. 이미 모듈에 내장된 경우 추가 불필요.

| VCC | 풀업 저항 값 |
|-----|-----------|
| 3.3 V | 4.7 kΩ |
| 5.0 V | 4.7 kΩ ~ 10 kΩ |

### LCD 1602 I2C 어댑터

PCF8574가 LCD 어댑터 모듈에 내장된 경우:
```
ESP32-C3 G6(SDA) → LCD 어댑터 SDA
ESP32-C3 G7(SCL) → LCD 어댑터 SCL
5V               → LCD 어댑터 VCC
GND              → LCD 어댑터 GND
```

## 제어 방법 (아두이노 코드)

### Wire 라이브러리 직접 사용

```cpp
#include <Wire.h>

const int PCF8574_ADDR = 0x20;
const int SDA_PIN = 6;
const int SCL_PIN = 7;

void setup() {
  Wire.begin(SDA_PIN, SCL_PIN);
}

// 8비트 전체 출력
void pcf8574Write(byte data) {
  Wire.beginTransmission(PCF8574_ADDR);
  Wire.write(data);
  Wire.endTransmission();
}

// 8비트 전체 읽기
byte pcf8574Read() {
  Wire.requestFrom(PCF8574_ADDR, 1);
  if (Wire.available()) return Wire.read();
  return 0xFF;
}

// 특정 핀만 OUTPUT HIGH/LOW (다른 핀 상태 유지)
byte portState = 0xFF;  // 전역 포트 상태 변수

void setPinOutput(int pin, bool value) {
  if (value) portState |=  (1 << pin);
  else       portState &= ~(1 << pin);
  pcf8574Write(portState);
}

// 특정 핀 입력 읽기 (먼저 해당 핀을 1로 설정)
bool readPin(int pin) {
  portState |= (1 << pin);   // 입력 핀은 1로 설정
  pcf8574Write(portState);
  byte data = pcf8574Read();
  return (data >> pin) & 0x01;
}

void loop() {
  // P0 ~ P7 순차 ON/OFF (LED 연결 시 Active LOW)
  for (int i = 0; i < 8; i++) {
    pcf8574Write(~(1 << i));  // 해당 핀만 LOW
    delay(200);
  }
  pcf8574Write(0xFF);  // 전체 OFF
  delay(500);
}
```

### PCF8574 라이브러리 사용

```cpp
#include <PCF8574.h>  // robtillaart/PCF8574 라이브러리

PCF8574 pcf(0x20, 6, 7);  // 주소, SDA, SCL

void setup() {
  pcf.begin();
  pcf.write8(0xFF);     // 전체 HIGH (입력 모드)
}

void loop() {
  pcf.write(0, LOW);    // P0 LOW
  delay(200);
  pcf.write(0, HIGH);   // P0 HIGH
  delay(200);

  bool btn = pcf.read(3);  // P3 읽기
}
```

### 인터럽트 활용

```cpp
const int INT_PIN = 3;
volatile bool changed = false;

void IRAM_ATTR pcfISR() {
  changed = true;
}

void setup() {
  Wire.begin(6, 7);
  pcf8574Write(0xFF);  // 전체 입력 모드
  pinMode(INT_PIN, INPUT_PULLUP);
  attachInterrupt(digitalPinToInterrupt(INT_PIN), pcfISR, FALLING);
}

void loop() {
  if (changed) {
    changed = false;
    byte state = pcf8574Read();
    Serial.printf("포트 상태: 0x%02X\n", state);
  }
}
```

## 주의사항

1. **소스 전류 한계**: PCF8574는 출력 HIGH 시 300µA밖에 공급 불가. LED를 HIGH Active로 직접 구동 불가. **LOW Active** 방식(출력 LOW 시 LED ON)으로 연결하고 VCC에서 전류 공급.
2. **풀업 저항 필수**: I2C SDA/SCL에 풀업 저항(4.7kΩ @ 3.3V) 없으면 통신 불가.
3. **주소 충돌**: 같은 버스에 PCF8574와 PCF8574A 혼용 가능 (주소 범위 다름). 단, SSD1306 OLED(0x3C/0x3D)와 PCF8574A 주소 겹칠 수 있음.
4. **입력 사용 전 1 쓰기**: 핀을 입력으로 사용하기 전에 반드시 해당 핀에 '1'을 써서 내부 풀업 활성화.
5. **포트 상태 관리**: Read-Modify-Write 시 최신 포트 상태를 소프트웨어로 추적 필요. 전역 변수로 현재 상태 유지.
6. **속도 제한**: 400 kHz Fast-mode 지원. 하지만 ESP32 I2C 속도 설정 확인 (`Wire.setClock(400000)`).
7. **전압 레벨**: PCF8574는 5V 동작 가능. ESP32-C3(3.3V)와 연결 시 VCC=3.3V로 설정하거나 레벨 시프터 사용.

## 자주 쓰이는 회로 패턴

### 패턴 1: 4×4 키패드 스캔 (8핀 모두 활용)

```cpp
// P0~P3: 행(Row) 출력, P4~P7: 열(Col) 입력
void scanKeypad() {
  for (int row = 0; row < 4; row++) {
    byte mask = ~(1 << row) | 0xF0;  // 해당 행만 LOW, 열은 모두 HIGH(입력)
    pcf8574Write(mask);
    delay(1);
    byte state = pcf8574Read();
    for (int col = 0; col < 4; col++) {
      if (!(state & (1 << (col + 4)))) {
        Serial.printf("키 눌림: 행%d 열%d\n", row, col);
      }
    }
  }
}
```

### 패턴 2: LCD 1602 I2C 제어 (PCF8574 직접)

```cpp
// PCF8574 핀 매핑: P0=RS, P1=RW, P2=E, P3=Backlight, P4~P7=D4~D7
void lcdSend(byte data, bool rsHigh) {
  byte high = (data & 0xF0) | (rsHigh ? 0x01 : 0x00) | 0x08;  // RS + BL
  byte low  = ((data << 4) & 0xF0) | (rsHigh ? 0x01 : 0x00) | 0x08;
  pcf8574Write(high | 0x04);  // E=HIGH
  delayMicroseconds(1);
  pcf8574Write(high);         // E=LOW
  pcf8574Write(low | 0x04);
  delayMicroseconds(1);
  pcf8574Write(low);
}
```

### 패턴 3: 2개 PCF8574 연결 (16개 GPIO)

```cpp
const int PCF_A = 0x20;  // A0=A1=A2=GND
const int PCF_B = 0x21;  // A0=VCC, A1=A2=GND

void writeAll16(uint16_t data) {
  Wire.beginTransmission(PCF_A);
  Wire.write((byte)(data & 0xFF));
  Wire.endTransmission();
  Wire.beginTransmission(PCF_B);
  Wire.write((byte)(data >> 8));
  Wire.endTransmission();
}
```
