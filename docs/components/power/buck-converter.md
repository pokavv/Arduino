# 벅(강압) 컨버터 모듈

## 개요

벅 컨버터(Buck Converter)는 스위칭 방식으로 높은 입력 전압을 낮은 출력 전압으로 변환하는 DC-DC 강압 컨버터입니다.
선형 레귤레이터(LM7805, AMS1117)가 차압을 열로 소모하는 것과 달리,
벅 컨버터는 인덕터에 에너지를 저장·방출하는 방식으로 전환하여 효율이 85~95%에 달합니다.

Arduino 프로젝트에서는 주로 다음 용도로 사용합니다.
- 12V/24V 배터리 또는 어댑터 → 5V/3.3V 변환 (배터리 구동 로봇, IoT 기기)
- 리튬 배터리 팩(11.1V) → Arduino 5V 구동
- 차량 12V → Arduino/센서 전원 공급
- 태양광 패널 전압 → 충전 및 시스템 전원

## 모듈 종류 비교

### MP1584EN 기반 소형 모듈 (3A 미니 벅)

- 크기: 약 22 × 17mm
- 입력: 4.5~28V
- 출력: 0.8~20V (가변), 최대 3A
- 조정: 소형 파란 트리머(가변 저항)
- 특징: 매우 소형, 리플 노이즈 적음, 고주파(1.5MHz) 덕분에 인덕터 소형화

### LM2596 기반 표준 모듈 (3A 표준 벅)

- 크기: 약 43 × 21mm
- 입력: 4.5~40V
- 출력: 1.25~35V (가변), 최대 3A
- 조정: 파란 트리머
- 특징: 저렴, 널리 유통, 방열판 있는 제품도 있음, 전압계 내장 버전 존재

### LM2596 전압 표시 모듈

- 크기: 약 53 × 25mm
- 기능: 입출력 전압 수치 표시 (4자리 LED 디스플레이)
- 특징: 직관적, 조정 편리, 가격 약간 높음

## 동작 원리

벅 컨버터는 4가지 핵심 소자로 구성됩니다.

```
Vin ── [스위치(MOSFET)] ── L(인덕터) ── Vout
              │                   │
         [다이오드]              [C(캐패시터)]
              │                   │
             GND                 GND
```

**동작 사이클**:
1. **ON 구간**: 스위치(MOSFET) ON → 전류가 Vin → L → Vout로 흐르며 인덕터에 에너지 저장
2. **OFF 구간**: 스위치 OFF → 인덕터에 저장된 에너지가 다이오드를 통해 Vout로 방출
3. 듀티비(ON 시간 / 전체 주기)를 조절하여 출력 전압 제어

```
Vout = Vin × 듀티비(D)
예) Vin=12V, D=0.417 → Vout ≈ 5V
```

효율이 높은 이유: 스위치는 ON/OFF만 하므로 선형 레귤레이터처럼 큰 전압차를 열로 소모하지 않습니다.

## 아두이노 연결 방법

### 기본 전원 공급

```
12V 전원 ──── 벅 컨버터 IN+ / IN−
              벅 컨버터 OUT+ ──── Arduino VIN (또는 5V 핀)
              벅 컨버터 OUT− ──── Arduino GND
```

### 전압 설정 방법

1. 벅 컨버터의 OUT+, OUT− 단자에 멀티미터 연결 (DC 전압 측정 모드)
2. 입력 전원 공급 (부하 없이)
3. 트리머를 드라이버로 돌리며 원하는 전압으로 조정
   - 시계 방향: 출력 전압 상승
   - 반시계 방향: 출력 전압 하강
4. 목표 전압보다 약간 높게(+0.1V) 설정 후 부하 연결 (부하 시 약간 전압 강하)

### 3.3V ESP32-C3 용도

```
입력: 5V USB 허브 또는 리튬 배터리
출력: 3.3V 설정 → ESP32-C3 3V3 핀에 연결

주의: 5V → 3.3V 변환 시 효율 약 82%
     소전류(< 500mA)에서는 LDO(AMS1117)가 더 적합할 수 있음
     고전류(> 500mA) 또는 고전압 입력 시 벅 컨버터가 유리
```

### 이중 전원 공급 회로

```
24V 전원 ─┬─ 벅 컨버터 A: 24V → 12V (서보모터, 스텝모터)
           ├─ 벅 컨버터 B: 24V → 5V  (Arduino, 릴레이, 센서)
           └─ 벅 컨버터 C: 24V → 3.3V (ESP32, 무선 모듈)
```

## 설정 방법

### 출력 전압 세밀 조정 순서

1. 멀티미터를 벅 컨버터 출력 단자에 연결
2. 부하(Arduino 등) 미연결 상태로 입력 전원 공급
3. 트리머를 천천히 조정 (트리머 회전수 10~20회 분량, 정밀 조정 필요)
4. 목표 전압 도달 시 멈춤
5. 부하 연결 후 전압 재확인 (일반적으로 0.05~0.1V 강하)
6. 필요 시 미세 조정

### 전류 제한 버전 모듈 (CC/CV 모듈)

일부 모듈은 전압(CV)과 전류(CC)를 모두 조정할 수 있습니다.
- CV 트리머: 출력 전압 조정
- CC 트리머: 최대 출력 전류 조정 (배터리 충전 회로에 활용)

## 주의사항

1. **출력 전압 확인 후 연결**: 모듈 기본값이 원하는 전압과 다를 수 있습니다. 반드시 멀티미터로 확인 후 부하를 연결하세요.
2. **입출력 차압 유지**: LM2596은 최소 1.5V, MP1584는 최소 0.7V의 입출력 차압이 필요합니다.
3. **전자기 간섭(EMI)**: 스위칭 주파수로 인해 라디오, 무선 모듈 근처에 배치 시 간섭 발생 가능. RF 모듈(nRF24L01, HC-05 등)과 공간적으로 분리하거나 차폐 필요.
4. **인덕터 자기장**: 가까이 있는 홀 센서, 나침반 모듈이 오동작할 수 있습니다.
5. **고주파 리플**: 아날로그 센서(ADC) 정확도에 영향. 벅 컨버터 출력 후단에 100μF + 100nF 캐패시터 추가로 개선.
6. **부하 없을 때 주의**: 일부 모듈은 무부하 상태에서 출력 전압이 불안정하거나 과전압이 발생할 수 있습니다.
7. **방향 주의**: IN+/IN−, OUT+/OUT− 극성을 반드시 확인. 역연결 시 즉시 파손.

## 실사용 팁 및 회로 패턴

### 선형 레귤레이터 vs 벅 컨버터 선택 기준

| 항목 | 선형 레귤레이터 | 벅 컨버터 |
|------|----------------|-----------|
| 효율 | 낮음 (30~70%) | 높음 (80~95%) |
| 발열 | 많음 | 적음 |
| 노이즈 | 매우 낮음 | 높음 (스위칭 리플) |
| 가격 | 저렴 | 약간 비쌈 |
| 크기 | 소형 가능 | 인덕터로 인해 약간 큼 |
| 복잡도 | 단순 | 보통 |
| 입출력 차압 | 클수록 효율↓ 발열↑ | 차압 무관 고효율 |
| 권장 상황 | 소전류(<200mA), RF 민감 회로 | 고전류, 배터리 구동, 고전압 입력 |

### 출력 노이즈 저감

```cpp
// 아날로그 측정 정확도 향상을 위한 필터 캐패시터 추가
// 벅 컨버터 OUT+ ─── [100μF 전해] ─── GND
//                 ─── [100nF 세라믹] ─── GND
//                 ─── ADC 핀

// 소프트웨어 오버샘플링으로 추가 개선
int oversample(int pin, int samples = 16) {
    long sum = 0;
    for (int i = 0; i < samples; i++) {
        sum += analogRead(pin);
    }
    return sum / samples;
}
```

### 배터리 구동 로봇 전원 설계

```
리튬 배터리 3S (11.1V) ─┬─ 벅 A: 11.1V → 5V/3A → Arduino + 서보
                         └─ 벅 B: 11.1V → 3.3V/1A → ESP32, 센서

전압 모니터링:
  배터리(+) → 분압 저항(100kΩ + 33kΩ) → Arduino A0
  코드에서 배터리 전압 11.1×(1/4) = 2.775V 검출
  cut-off: 배터리 셀당 3.0V = 전체 9.0V → 경고 LED 점멸
```
