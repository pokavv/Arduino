# 서보모터 (Servo Motor)

## 개요

서보모터는 위치 제어가 가능한 회전형 액추에이터입니다. 내부에 DC 모터, 감속 기어, 포텐셔미터(위치 피드백), 제어 회로가 일체화되어 있으며, PWM 신호를 통해 정해진 각도로 정밀하게 회전합니다. 아두이노 프로젝트에서 로봇 팔, 문/뚜껑 개폐, 카메라 짐벌, RC 차량 조향 등에 널리 사용됩니다.

**표준 서보(Standard Servo)**: 0~180° 범위 제어  
**연속 회전 서보(Continuous Rotation Servo)**: 속도·방향 제어 (각도 제어 불가)

## 핀/단자 구성

서보모터는 3선 커넥터를 사용합니다.

| 선 색상 | 핀 명칭 | 기능 |
|---------|---------|------|
| 갈색 / 검정 | GND | 접지 (0V) |
| 빨강 | VCC | 전원 공급 (4.8~7.2V) |
| 주황 / 노랑 / 흰색 | Signal | PWM 제어 신호 입력 |

> 제조사마다 색상 배열이 다를 수 있음. 반드시 데이터시트 확인.

## 동작 원리

서보모터는 **폐루프 제어(Closed-loop Control)** 방식으로 동작합니다.

1. 제어 보드에서 50Hz PWM 신호를 서보에 전달
2. 내부 제어 IC가 펄스 폭을 측정하여 목표 각도를 계산
3. 내부 포텐셔미터로 현재 출력축 각도를 피드백
4. 목표 각도와 현재 각도 오차만큼 DC 모터 구동
5. 목표 각도에 도달하면 모터 정지 (스톨 상태에서 위치 유지)

**펄스 폭 ↔ 각도 대응:**
- 펄스 1000 µs → 0° (좌측 끝)
- 펄스 1500 µs → 90° (중앙)
- 펄스 2000 µs → 180° (우측 끝)

실제 각도 범위는 제조사마다 다를 수 있으므로 500~2500 µs 범위로 캘리브레이션 필요.

## 아두이노 연결 방법

### 소형 서보 (SG90, MG90S) — 아두이노 직접 구동 가능 (주의)

```
ESP32-C3 Super Mini        SG90 / MG90S
────────────────────       ─────────────
3.3V 또는 5V   ────────→   빨강 (VCC)     ← 가급적 외부 5V 전원
GND            ────────→   갈색 (GND)
G4 (또는 임의) ────────→   주황 (Signal)
```

> ESP32-C3는 3.3V GPIO 출력이므로 Signal 핀은 직결 가능.  
> 전원은 아두이노 5V 핀 사용 시 전류 부족 위험 → **외부 5V 전원 권장**.

### MG996R — 반드시 외부 전원

```
외부 5~6V 전원
├── (+) ────→ 서보 빨강 (VCC)
└── (-) ────→ 서보 갈색 (GND)
               서보 갈색 ────→ ESP32-C3 GND  (공통 GND 필수!)
ESP32-C3 G4 ──────────────→ 서보 주황 (Signal)
```

**공통 GND 연결 필수**: ESP32-C3 GND와 외부 전원 GND를 반드시 연결해야 PWM 신호 기준이 맞음.

### 전원 공급 가이드

| 서보 모델 | 권장 전원 공급 방법 |
|-----------|-------------------|
| SG90 | 아두이노 5V 핀 직접 (최대 1~2개) |
| MG90S | 아두이노 5V 핀 또는 외부 5V |
| MG996R | 외부 5~6V, 2A 이상 전원 공급기 필수 |
| 여러 개 동시 | 외부 전원 필수, 서보 수 × 600mA 이상 |

## 제어 방법

### Arduino Servo 라이브러리 사용

```cpp
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(4);          // G4 핀에 서보 연결
  // myServo.attach(4, 500, 2500);  // 커스텀 범위 (µs)
}

void loop() {
  myServo.write(0);    // 0도
  delay(1000);
  myServo.write(90);   // 90도
  delay(1000);
  myServo.write(180);  // 180도
  delay(1000);
}
```

### ESP32 ledc를 이용한 직접 PWM 제어 (더 정밀)

```cpp
// ESP32에서 Servo 라이브러리 없이 직접 PWM 제어
const int SERVO_PIN  = 4;
const int LEDC_CH    = 0;
const int LEDC_FREQ  = 50;   // 50 Hz
const int LEDC_RES   = 16;   // 16비트 해상도

// 각도를 16비트 듀티로 변환
uint32_t angleToDuty(int angle) {
  // 50Hz, 16비트: 20ms 주기 = 65536
  // 500µs = 1638, 2500µs = 8192
  return map(angle, 0, 180, 1638, 8192);
}

void setup() {
  ledcSetup(LEDC_CH, LEDC_FREQ, LEDC_RES);
  ledcAttachPin(SERVO_PIN, LEDC_CH);
}

void loop() {
  ledcWrite(LEDC_CH, angleToDuty(0));
  delay(1000);
  ledcWrite(LEDC_CH, angleToDuty(90));
  delay(1000);
  ledcWrite(LEDC_CH, angleToDuty(180));
  delay(1000);
}
```

### PWM 파라미터 정리

| 항목 | 값 |
|------|----|
| 주파수 | 50 Hz |
| 주기 | 20 ms |
| 0° 펄스 폭 | 500 ~ 1000 µs (서보별 상이) |
| 90° 펄스 폭 | 1500 µs |
| 180° 펄스 폭 | 2000 ~ 2500 µs |

## 주의사항

1. **전류 공급**: 아두이노 GPIO는 서보 신호 출력만 담당. 전원(VCC)은 반드시 충분한 전류 공급 가능한 소스에서 공급.
2. **공통 GND**: 외부 전원을 쓸 때 아두이노 GND와 외부 전원 GND를 반드시 연결.
3. **역기전력(Back-EMF)**: 서보가 급격히 멈추거나 과부하 시 전압 스파이크 발생 → 전원 라인에 100~470µF 전해 커패시터 추가 권장.
4. **기계적 한계 초과 금지**: 서보를 손으로 강제 회전시키거나, 소프트웨어에서 물리적 한계를 초과하는 각도 명령 금지 → 기어 파손.
5. **연속 스톨 금지**: 목표에 도달 후에도 지속적으로 힘이 가해지면 과열 및 소손. 작업 완료 후 `myServo.detach()`로 PWM 신호 제거 권장.
6. **ESP32 Servo 라이브러리**: ESP32 전용 `ESP32Servo` 라이브러리 사용 권장 (표준 Servo 라이브러리는 ESP32에서 불안정할 수 있음).

## 자주 쓰이는 회로 패턴

### 패턴 1: 포텐셔미터로 서보 제어

```
포텐셔미터 3.3V — (VCC)
포텐셔미터 GND  — (GND)
포텐셔미터 OUT  — G1 (ADC)

G1 ADC 읽기 → map(adcVal, 0, 4095, 0, 180) → 서보 각도
```

### 패턴 2: 버튼으로 서보 위치 토글

```cpp
if (digitalRead(BTN_PIN) == LOW) {
  servoAngle = (servoAngle == 0) ? 180 : 0;
  myServo.write(servoAngle);
}
```

### 패턴 3: 밀리초 기반 스위프 (논블로킹)

```cpp
unsigned long lastTime = 0;
int angle = 0;
int step  = 1;

void loop() {
  if (millis() - lastTime >= 15) {
    lastTime = millis();
    angle += step;
    if (angle >= 180 || angle <= 0) step = -step;
    myServo.write(angle);
  }
}
```

### 패턴 4: 여러 서보 동시 제어

```cpp
Servo servo1, servo2, servo3;

void setup() {
  servo1.attach(4);
  servo2.attach(5);
  servo3.attach(6);
}
```
