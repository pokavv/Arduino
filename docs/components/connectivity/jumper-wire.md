# 점퍼 와이어 (Jumper Wire)

## 개요

점퍼 와이어는 브레드보드, Arduino 보드, 센서 모듈 등을 납땜 없이 연결하는 단선형 전선입니다.
프로토타입 제작과 실험에서 없어서는 안 될 필수 연결 부품입니다.

Arduino 프로젝트에서의 활용:
- 브레드보드 ↔ Arduino 핀 연결
- 센서/모듈 ↔ Arduino 연결
- 전원 및 GND 공급 라인
- 디버그/임시 배선

---

## 종류 비교 표

| 종류 | 영문명 | 커넥터 구성 | 주요 용도 |
|------|--------|-----------|---------|
| 수-수 | Male to Male (M-M) | 양쪽 핀 | 브레드보드 ↔ 브레드보드, 브레드보드 내부 연결 |
| 수-암 | Male to Female (M-F) | 한쪽 핀, 한쪽 소켓 | Arduino 핀 → 브레드보드, 모듈 소켓 → 브레드보드 |
| 암-암 | Female to Female (F-F) | 양쪽 소켓 | Arduino 핀 ↔ 모듈 핀 헤더, Arduino ↔ Arduino |

---

## 핀 구성 (커넥터 상세)

### 수(Male) 커넥터

```
금속 핀(직경 약 0.64mm) ─── 플라스틱 하우징
```

- 핀 직경: 약 0.64mm (표준 2.54mm 피치용)
- 브레드보드 구멍, 소켓 헤더에 삽입
- 핀 길이(삽입부): 약 8~10mm

### 암(Female) 커넥터

```
소켓 하우징 ─── 내부 금속 클립(스프링 접촉) ─── 배선
```

- 핀 헤더(수 핀)를 수용
- 내부 접촉 클립이 스프링 압력으로 전기 접촉 유지

### AWG(American Wire Gauge) 규격 상세

| AWG | 직경 (mm) | 단면적 (mm²) | 저항 (Ω/m) | 최대 전류 |
|-----|----------|------------|----------|---------|
| 20 | 0.812 | 0.517 | 0.0336 | 1.5A |
| 22 | 0.644 | 0.326 | 0.0533 | 0.92A |
| 24 | 0.511 | 0.205 | 0.0847 | 0.577A |
| 26 | 0.405 | 0.129 | 0.1345 | 0.361A |
| 28 | 0.321 | 0.081 | 0.2139 | 0.226A |
| 30 | 0.255 | 0.051 | 0.3386 | 0.142A |

> Arduino 프로젝트의 일반 신호선(디지털 I/O, SPI, I2C)은 수 mA ~ 수십 mA이므로 28AWG로 충분합니다.
> 전원/GND 라인은 22~24AWG 이상 권장.

## 동작 원리

점퍼 와이어는 전기적으로 단순한 도선이지만, 두 가지 접촉 방식이 있습니다.

**단선(Solid Core)**: 구리선 1가닥. 브레드보드 삽입 시 마찰력이 일정하고 모양 유지가 쉬움. 반복 굴곡에 약함.

**연선(Stranded)**: 가는 구리선 여러 가닥. 유연하고 단선 저항 강함. 브레드보드에서 빠지기 쉬움.

일반 점퍼 와이어 세트는 보통 단선 또는 얇은 연선을 사용합니다.

## 아두이노 연결 방법

### 종류별 사용 시나리오

**M-M 와이어** (브레드보드 작업 기본):
```
브레드보드 전원 레일 → Arduino 5V/GND 핀
브레드보드 신호 홀 → Arduino 디지털/아날로그 핀
```

**M-F 와이어** (Arduino 핀 → 모듈):
```
Arduino 핀 헤더(수) → M-F 와이어 암(F) 끝 연결
→ M(수) 끝 → 브레드보드 삽입
또는
센서 모듈 핀 헤더(수) → M-F 암 끝 연결
→ M(수) 끝 → 브레드보드 삽입
```

**F-F 와이어** (모듈 간 직결):
```
Arduino 핀 헤더(수) → F-F 와이어 → 센서 모듈 핀 헤더(수)
(브레드보드 없이 직결)
```

### 색상 코딩 관례

| 색상 | 관례 |
|------|------|
| 빨강 | VCC, 전원 (+) |
| 검정 | GND, 전원 (−) |
| 노랑/주황 | SCK, CLK, 클럭 |
| 파랑 | SDA, 데이터 |
| 초록 | MISO, 수신 |
| 흰색 | MOSI, 송신 |
| 보라 | CS, SS, 칩 선택 |
| 회색 | 기타 신호 |

## 주의사항

1. **전류 용량 초과 금지**: 모터, 릴레이 코일 전류(수백 mA ~ 1A)를 28AWG 점퍼 와이어로 연결하면 과열 및 단선 위험. 22AWG 이상 또는 전용 전선 사용.
2. **브레드보드 접촉 불량**: 점퍼 와이어를 반복 삽입/제거하면 끝단 핀이 휘거나 소켓이 느슨해져 접촉 불량이 발생합니다. 의심 시 와이어 교체.
3. **길이에 따른 임피던스**: 고주파 신호(SPI, I2C)를 긴 점퍼 와이어로 연결하면 정전용량이 증가하여 신호 왜곡 가능. 가급적 짧게 연결.
4. **엉킴 방지**: 여러 와이어가 엉키면 회로 추적이 어렵습니다. 길이에 맞는 와이어 선택, 케이블 타이 또는 슬리브로 묶어 정리.
5. **단선(전선 내부 단절)**: 외관상 멀쩡해 보여도 내부 단선이 발생한 와이어는 멀티미터로만 확인 가능. 원인 불명의 오동작 시 먼저 와이어를 의심.

## 실사용 팁 및 회로 패턴

### 디버깅용 멀티미터 연속성 검사

```
멀티미터 연속성(Continuity) 모드:
  프로브를 와이어 양쪽 커넥터에 접촉
  "삐" 소리 → 연결 정상
  무음 → 내부 단선 → 와이어 교체
```

### 와이어 길이 선택 기준

| 사용 위치 | 권장 길이 |
|----------|---------|
| 브레드보드 내부 연결 | 5~10cm |
| Arduino → 브레드보드 | 10~20cm |
| 외부 센서 연결 | 20~30cm |
| 원거리 연결 | 별도 전선 가공 권장 |

### 긴 거리 신호 전달 시 노이즈 대책

```
장거리 신호선은 GND 선과 꼬아서(Twisted Pair) 배선:
→ 공통 모드 노이즈 제거

또는 양단에 100~1000pF 캐패시터를 신호선-GND 간 삽입:
→ 고주파 노이즈 필터링
```

### 와이어 재활용 팁

```
단선된 점퍼 와이어:
1. 양쪽 커넥터를 5mm씩 자름
2. 피복을 벗겨 내부 단선 위치 확인
3. 단선 부위를 잘라내고 새 커넥터 압착 (크림핑 툴 필요)
또는
4. 그냥 버리고 새 것 사용 (저렴하므로 교체 권장)
```
