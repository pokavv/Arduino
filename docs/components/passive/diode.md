# 정류 다이오드 (Rectifier Diode)

## 개요

다이오드(Diode)는 전류를 한 방향으로만 흐르게 하는 반도체 소자입니다. P형 반도체와 N형 반도체를 접합한 구조(PN 접합)로, 전류는 **애노드(Anode, +) → 캐소드(Cathode, -)** 방향으로만 흐릅니다.

**기호**: D  
**주요 특성**: 순방향 도통, 역방향 차단

1N4007은 가장 많이 사용되는 범용 정류 다이오드로, 내압 1000V, 최대 전류 1A의 스펙을 가집니다. 아두이노 프로젝트에서는 역극성 보호, 전원 정류, 환류 다이오드(릴레이/모터 보호) 등에 필수적으로 사용됩니다.

---

## 종류

### 1. 정류 다이오드 (Rectifier Diode) — 본 문서
- 교류를 직류로 변환 (정류 회로)
- 순방향 전압 강하: 약 0.6 ~ 0.7V (실리콘)
- **1N4001 ~ 1N4007** 시리즈: 가장 범용적
- 역극성 보호, 환류 다이오드, 단순 정류에 사용

### 2. 쇼트키 다이오드 (Schottky Diode)
- 순방향 전압 강하 낮음: 0.15 ~ 0.45V
- 스위칭 속도 빠름
- **1N5817 ~ 1N5819** 시리즈, **SS14 (SMD)**
- DC-DC 컨버터, 고속 스위칭, 저전압 회로에 적합
- 역내압이 상대적으로 낮음

### 3. 신호 다이오드 (Signal Diode)
- 소전류용, 고속 스위칭
- **1N4148**: 가장 흔한 신호 다이오드 (200mA, 100V)
- 로직 회로, 클램핑, 검파에 사용

### 4. 발광 다이오드 (LED)
- 전류 흐를 때 빛 방출
- 빨강: Vf ≈ 2.0V, 파랑/흰색: Vf ≈ 3.0~3.5V

### 5. 포토다이오드 (Photodiode)
- 빛을 받으면 역방향 전류 흐름
- 광센서, 광 통신에 사용

### 6. 브릿지 정류기 (Bridge Rectifier)
- 4개의 다이오드를 브릿지 형태로 조합
- 교류(AC)를 완전 직류(DC)로 변환
- **DB107**, **W10M** 등 패키지형 제품 있음

## 핀/단자 구성

### DO-41 패키지 (1N4007 표준)

```
   캐소드(K, -)   애노드(A, +)
        │              │
   ─────┤    ┌────┐    ├─────
        │    │ D  │    │
        └────┴────┴────┘
        회색/은색 띠 = 캐소드(-)

회로 기호:
     A(+)    K(-)
      │       │
      ▼ ── ──┤
    애노드  캐소드
  전류 흐름 방향 →
```

**극성 구분**:
- **은색/회색 띠(띠 있는 쪽)** = 캐소드(-) = 전류 출구
- **띠 없는 쪽** = 애노드(+) = 전류 입구
- 전류는 애노드(+) → 캐소드(-) 방향으로만 흐름

## 동작 원리

### PN 접합 동작

**순방향 바이어스 (Forward Bias)**:
```
+ 전압 → 애노드(P형) | 접합 | 캐소드(N형) ← - 전압

전위장벽(약 0.6V) 이상의 전압 인가 시 전류 흐름
실리콘: 약 0.6 ~ 0.7V
게르마늄: 약 0.2 ~ 0.3V (현재 거의 미사용)
```

**역방향 바이어스 (Reverse Bias)**:
```
- 전압 → 애노드 | 접합 | 캐소드 ← + 전압

공핍층이 넓어져 전류 차단 (소량의 누설 전류만 흐름)
역방향 내압(항복 전압) 이상 인가 시 파괴
```

### 전류-전압 특성 (I-V Curve)

```
      전류(I)
        │
        │         ← 순방향 (지수함수적 증가)
        │        /
        │       /
        │      /
────────┼─────/──────── 전압(V)
        │    0.6V
       /│
역방향  │  ← 역방향 (거의 0)
      ──│
항복점  │  ← 항복 전압 이상: 파괴
```

## 아두이노 회로 연결 방법

### 1. 역극성 보호 (전원 보호)

```
방법 1: 직렬 연결
[전원 +] ──[A|D|K]──── VCC
                         아두이노
[전원 -] ──────────── GND

효과: 전원 극성이 바뀌어도 회로 보호
단점: 0.7V 전압 강하 발생 (5V → 4.3V)
```

```
방법 2: P채널 MOSFET 사용 시 더 효율적 (전압 강하 최소화)
아두이노 입문자는 방법 1이 간단함
```

### 2. 환류 다이오드 (Flyback Diode) — 릴레이 보호

```
[VCC 5V] ────────┬──── 릴레이 코일(+)
                 │
              [1N4007] ← 캐소드를 VCC 방향으로
              (K→VCC, A→코일+)
                 │
[GPIO]─[트랜지스터]─ 릴레이 코일(-) ─── GND

역할: 릴레이 코일에서 발생하는 역기전력(Back-EMF) 흡수
필수: 이것 없으면 아두이노 핀 또는 트랜지스터 파괴
```

### 3. AC-DC 반파 정류

```
[AC 입력]
    │
   [변압기]
    │
    ├── [1N4007] ──┬── DC 출력(+)
    │     (A→K)   │
    │             [C: 1000μF]
    │             │
    └─────────────┴── GND(-)

출력은 맥류(맥동 직류)
커패시터로 평활화 필요
```

### 4. 전압 클램핑 (신호 보호)

```
GPIO ──┬──────── 신호 출력
       │
    [1N4148] ← 캐소드를 3.3V에
       │
      3.3V

효과: 신호가 3.3V+0.7V = 4V 이상이 되지 않도록 제한
(5V 신호를 ESP32 3.3V 입력에 연결 시 보호 용도)
```

### 5. OR 게이트 (다이오드 OR)

```
입력A ──[D1]──┬── 출력
입력B ──[D2]──┘
              │
           [R: 10kΩ]
              │
             GND

A 또는 B 중 하나라도 HIGH이면 출력 HIGH
두 전원을 OR로 선택할 때도 사용 (우선순위 전원 선택)
```

## 마킹 읽는 법

### 1N4007 외형 식별

```
     ┌────────────────┐
─────┤                ├─────
     └────────────────┘
                      │
               은색 띠 = 캐소드(-)

본체에 "1N4007" 또는 "4007" 인쇄
띠(Band) 있는 쪽 = 캐소드(-)
```

### SMD 다이오드 마킹

**SOD-123 / SOD-323 패키지 (신호용)**:
- `T4` = 1N4148
- `A7` = 1N4007 SMD 버전 (M7)
- `SS14` = 쇼트키 1A, 40V
- `B5819W` = 쇼트키 1N5819 SMD

**브릿지 정류기**:
- 4개 핀: AC~, AC~, +, -
- 패키지에 + 표시 또는 핀 순서 확인 필요

## 주의사항

### 1. 극성 역연결 금지
- 역방향 내압 이상의 전압을 역으로 인가하면 파괴
- **띠(Band) 있는 쪽이 항상 캐소드(-)**임을 확인

### 2. 전류 한계 준수
- 1N4007 최대 전류: 1A (평균)
- 서지 전류: 30A (1사이클 한정)
- 장시간 정격 전류 이상 흐르면 과열로 파괴

### 3. 순방향 전압 강하 고려
- 0.6~0.7V 전압 강하 발생
- 5V → 다이오드 → 4.3V (LED 밝기 저하 등 고려)
- 저전압 회로(3.3V)에서는 쇼트키 다이오드(0.3V 강하) 권장

### 4. 열 방출
- 고전류 회로에서는 방열판(Heatsink) 또는 충분한 공간 필요
- 1A 지속 통전 시 패키지 온도 상승 주의

### 5. 스위칭 속도
- 1N4007은 역회복 시간(trr)이 30μs로 느림
- 고주파 스위칭(DC-DC 컨버터 등)에는 쇼트키 다이오드 사용
- 오디오 주파수(60Hz 정류)에는 문제없음

### 6. 방향 확인 방법 (멀티미터)
- 멀티미터 다이오드 모드에서 약 0.6V 표시되면 순방향
- 역방향은 OL(개방) 또는 매우 높은 값

## 데이터시트 주요 파라미터 설명

| 파라미터 | 영문명 | 설명 |
|----------|--------|------|
| 역방향 반복 내압 | VRRM (Peak Repetitive Reverse Voltage) | 반복적으로 인가 가능한 최대 역방향 피크 전압 |
| 역방향 RMS 전압 | VRMS (RMS Reverse Voltage) | 역방향 RMS 전압 최대값 |
| 평균 순방향 전류 | IF(AV) or IO | 반파 정류 조건에서의 평균 순방향 전류 |
| 반복 피크 순방향 전류 | IFRM | 반복 허용 피크 순방향 전류 |
| 비반복 서지 전류 | IFSM | 1사이클 동안 허용되는 최대 서지 전류 |
| 순방향 전압 강하 | VF (Forward Voltage) | 지정 전류에서의 순방향 전압 강하 |
| 역방향 누설 전류 | IR (Reverse Current) | 역방향 전압 인가 시 흐르는 소량의 전류 |
| 역회복 시간 | trr (Reverse Recovery Time) | 역방향 전류에서 차단까지 걸리는 시간 |
| 접합 커패시턴스 | Cj (Junction Capacitance) | 역방향 바이어스 시의 접합 커패시턴스 |
| 접합 온도 | TJ (Junction Temperature) | 최대 허용 접합 온도 |
| 열 저항 | RθJA (Thermal Resistance) | 접합에서 주위 공기까지의 열 저항 |
