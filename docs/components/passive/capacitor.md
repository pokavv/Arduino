# 커패시터 (Capacitor)

## 개요

커패시터(Capacitor, 콘덴서)는 전하를 저장하는 수동 소자입니다. 두 개의 도체판 사이에 절연체(유전체)를 끼운 구조로, 전압을 일시적으로 저장하거나 교류(AC)를 통과시키고 직류(DC)를 차단하는 특성이 있습니다.

**단위**: 패럿(F, Farad) — 실용 단위는 μF(마이크로패럿), nF(나노패럿), pF(피코패럿)  
**기호**: C  
**기본식**: Q = C × V (전하량 = 커패시턴스 × 전압)

아두이노 프로젝트에서는 전원 디커플링(노이즈 제거), 버튼 디바운싱 필터, RC 필터, 파워 서플라이 안정화 등에 사용됩니다.

### 단위 변환

| 단위 | 표기 | 값 |
|------|------|----|
| 1 패럿 | 1 F | 1 F |
| 1 밀리패럿 | 1 mF | 10⁻³ F |
| 1 마이크로패럿 | 1 μF | 10⁻⁶ F |
| 1 나노패럿 | 1 nF | 10⁻⁹ F |
| 1 피코패럿 | 1 pF | 10⁻¹² F |

## 종류

### 1. 전해 커패시터 (Electrolytic Capacitor)
- **극성 있음** (+ / - 구분 필수)
- 대용량 (1μF ~ 47,000μF), 저렴함
- 전압 안정화, 전원 필터링, 대용량 에너지 저장에 사용
- 수명이 있음 (고온에서 단축), 역극성 연결 시 폭발 위험
- 알루미늄 전해, 탄탈 전해 등

### 2. 세라믹 커패시터 (Ceramic Capacitor)
- **극성 없음**, 소형, 저렴
- 소용량 (1pF ~ 100μF)
- 고주파 특성 우수, 디커플링·바이패스에 최적
- 온도/전압에 따라 용량 변화 있음 (등급에 따라 다름)
- **C0G/NP0**: 온도 안정적, 고정밀 — pF ~ 수십 nF
- **X5R/X7R**: 일반용, 100nF ~ 수십 μF
- **Y5V/Z5U**: 저가, 대용량이지만 특성 불안정

### 3. 필름 커패시터 (Film Capacitor)
- **극성 없음**, 정밀도 높음
- 폴리에스터(Polyester), 폴리프로필렌(Polypropylene) 등
- 오디오 회로, 타이밍 회로, 고정밀 필터에 적합
- 탄탈 커패시터보다 안정적

### 4. 탄탈 커패시터 (Tantalum Capacitor)
- **극성 있음** (+ 표시 확인 필수)
- 소형 고용량, 전해보다 수명 길고 특성 안정
- 역전압에 매우 취약 (폭발/발화 위험)
- SMD 형태로 많이 사용

### 5. 슈퍼커패시터 (Supercapacitor / EDLC)
- 매우 큰 용량 (1F ~ 수천F)
- 배터리 보조, 에너지 하베스팅, 순간 백업 전원
- 낮은 에너지 밀도, 높은 전력 밀도

### 6. SMD 커패시터
- 표면 실장형, 0402 / 0603 / 0805 / 1206 등 크기 규격
- 세라믹, 탄탈 등 다양한 종류

## 핀/단자 구성

### 전해 커패시터 (극성 있음)

```
      +    -
      |    |
  ────┴────┴────
  리드 긴 쪽이 +극 (양극)
  본체 흰색/회색 줄무늬 쪽이 -극 (음극)
  
     ┌──────┐
     │  (-)  │  ← 흰색 띠 = 음극
     │  전해  │
     │       │
     └──┬──┬─┘
        │  │
       (+) (-)
```

**극성 구분**:
- **긴 리드** = + (양극, Anode)
- **짧은 리드** = - (음극, Cathode)
- 본체의 흰색/회색 **세로 줄무늬** = - (음극)

### 세라믹 커패시터 (극성 없음)

```
     ┌──────┐
─────┤      ├─────
     └──────┘
어느 방향으로 연결해도 무방
```

### SMD 탄탈 커패시터 (극성 있음)

```
  ┌─────┐
  │ + 표시│  ← + 표시된 쪽이 양극
  └─────┘
```

## 동작 원리

### 충전과 방전

1. **충전**: 전압을 가하면 두 도체판에 서로 반대 부호의 전하가 쌓임
2. **방전**: 전압을 제거하면 축적된 전하가 방출되어 전류 흐름

```
충전 시:
전원(+) ──→ 판(+) | 유전체 | 판(-) ──→ 전원(-)

방전 시:
판(+) ──→ 부하 ──→ 판(-)
```

### 커패시터의 주요 특성

- **DC 차단**: 직류는 통과시키지 않음 (충전 후 전류 차단)
- **AC 통과**: 교류는 연속적으로 충방전하여 전류가 흐르는 것처럼 동작
- **임피던스**: Xc = 1 / (2πfC) — 주파수가 높을수록 임피던스 낮아짐
- **에너지 저장**: E = ½CV²

### RC 시상수 (Time Constant)

```
τ (타우) = R × C

- 1τ: 충전율 63.2% 도달
- 3τ: 충전율 95% 도달
- 5τ: 충전율 99.3% (완전 충전으로 간주)

예: R=10kΩ, C=10μF → τ = 10,000 × 0.00001 = 0.1초 = 100ms
```

## 아두이노 회로 연결 방법

### 1. 전원 디커플링 (Decoupling)

```
3.3V ──┬────── IC VCC
       │
      [100nF] 세라믹 커패시터
       │
GND ───┴────── IC GND

목적: IC 동작 시 발생하는 전원 노이즈 흡수
위치: 가능한 IC VCC 핀에 최대한 가깝게 배치 (2cm 이내)
추가: 10μF 전해 커패시터를 병렬로 추가하면 더 효과적
```

### 2. 버튼 디바운싱 필터

```
3.3V ──[10kΩ]──┬── GPIO 핀
               │
              [100nF]
               │
GND ──[버튼]───┘

효과: 버튼 접점 진동(바운스)로 인한 잘못된 입력 방지
대안: 소프트웨어 디바운싱 (delay() 또는 millis() 사용)
```

### 3. RC 저역통과 필터 (Low-Pass Filter)

```
입력 ──[R: 10kΩ]──┬── 출력 (ADC)
                  │
                 [C: 100nF]
                  │
                 GND

차단 주파수: fc = 1 / (2π × R × C) = 1 / (2π × 10000 × 0.0000001) ≈ 159 Hz
용도: 아날로그 입력 노이즈 제거
```

### 4. 전원 안정화 (파워 서플라이)

```
VCC ──────┬───────────────── VCC (안정화)
          │
        [100μF] 전해 (대용량)
          │
          │
        [100nF] 세라믹 (고주파 필터)
          │
GND ──────┴───────────────── GND
```

### 5. 모터/릴레이 전원 분리

```
[배터리/어댑터]
       │
 +────┬┴────── 모터 전원
      │
    [470μF] ← 모터 기동 전류 공급
      │
 -────┴──────── GND

목적: 모터 기동 시 전압 강하로 MCU 리셋 방지
```

## 표기법 읽는 법

### 세라믹/필름 커패시터 코드 표기

**3자리 숫자 코드** (단위: pF)
- 앞 두 자리 = 유효 숫자
- 세 번째 자리 = 10의 지수

| 코드 | 계산 | 실제 값 |
|------|------|---------|
| 104 | 10 × 10⁴ | 100,000 pF = 100 nF = 0.1 μF |
| 473 | 47 × 10³ | 47,000 pF = 47 nF |
| 102 | 10 × 10² | 1,000 pF = 1 nF |
| 221 | 22 × 10¹ | 220 pF |
| 100 | 10 × 10⁰ | 10 pF |
| 010 | 10 pF (그대로) | 10 pF |

**허용 오차 문자 코드**

| 문자 | 오차 |
|------|------|
| B | ±0.1 pF |
| C | ±0.25 pF |
| D | ±0.5 pF |
| F | ±1% |
| G | ±2% |
| J | ±5% |
| K | ±10% |
| M | ±20% |
| Z | +80% / -20% |

**예시**: `104K` → 100nF, ±10%

### 전해 커패시터 표기

직접 수치 기입:
- `100μF 25V` — 100마이크로패럿, 내압 25V
- `470μF 16V 105°C` — 470μF, 16V 내압, 최대 105°C

### SMD 커패시터

크기가 작아 마킹이 없거나 매우 간략함. 릴(Reel) 테이프의 라벨로 구분.

## 주의사항

### 1. 극성 역연결 절대 금지 (전해, 탄탈)
- **전해 커패시터**를 역극성으로 연결하면 내부 가스 발생 → **폭발** 위험
- 탄탈 커패시터는 역극성 시 **발화** 가능
- 반드시 + 리드(긴 쪽)를 고전위, - 리드(짧은 쪽)를 저전위에 연결

### 2. 정격 전압 초과 금지
- 정격 전압보다 높은 전압을 가하면 유전체 파괴 → 단락 또는 폭발
- 안전 마진: 정격 전압의 **60~80% 이하**에서 사용 권장
- 예: 16V 정격 → 실제 사용 전압은 10V 이하로 유지

### 3. 온도 주의 (전해 커패시터)
- 85°C 등급과 105°C 등급이 있음
- 고온 환경에서는 반드시 105°C 등급 사용
- 발열 부품(전압 레귤레이터 등) 근처 배치 금지

### 4. 세라믹 커패시터 전압 계수
- X5R, X7R 등급 세라믹 커패시터는 전압이 높아지면 실제 용량이 크게 감소
- 예: 10V 정격 10μF X5R 커패시터를 5V에서 사용하면 실제 용량은 5μF 수준으로 감소 가능
- 대책: 정격 전압 대비 실제 사용 전압 비율을 낮추거나 C0G/NP0 사용

### 5. 직렬/병렬 연결
- **직렬**: 합성 용량 감소 (1/C = 1/C1 + 1/C2), 내압 증가
- **병렬**: 합성 용량 증가 (C = C1 + C2), 내압은 가장 낮은 것 기준

### 6. 방전 위험
- 큰 용량의 커패시터(슈퍼커패시터 등)는 충전 후 큰 전류를 순간적으로 방출 가능
- 맨손으로 단자를 동시에 만지지 말 것

## 데이터시트 주요 파라미터 설명

| 파라미터 | 영문명 | 설명 |
|----------|--------|------|
| 정전 용량 | Capacitance (C) | 기본 용량값, 단위 F |
| 허용 오차 | Tolerance | 실제값의 오차 범위 |
| 정격 전압 | Rated Voltage (WV) | 안전하게 사용할 수 있는 최대 전압 |
| 내전압 | Surge/Peak Voltage | 단기 허용 최대 전압 (정격의 1.2~2배) |
| 등가 직렬 저항 | ESR (Equivalent Series Resistance) | 내부 저항, 낮을수록 효율 높음 |
| 등가 직렬 인덕턴스 | ESL (Equivalent Series Inductance) | 내부 인덕턴스, 고주파 특성에 영향 |
| 누설 전류 | Leakage Current | 충전 후 자연 방전되는 전류 |
| 손실 계수 | Dissipation Factor (DF / tan δ) | 에너지 손실 비율 |
| 절연 저항 | Insulation Resistance (IR) | 커패시터 내부 절연 저항 |
| 리플 전류 | Ripple Current | 연속으로 흐를 수 있는 교류 성분 전류 최대값 |
| 공진 주파수 | Self Resonant Frequency (SRF) | 커패시터가 인덕터처럼 동작하기 시작하는 주파수 |
| 온도 계수 | Temperature Coefficient (TC) | 온도에 따른 용량 변화율 |
| 수명 | Load Life / Endurance | 정격 조건에서의 예상 수명 (시간) |
