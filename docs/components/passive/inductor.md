# 인덕터 (Inductor)

## 개요

인덕터(Inductor, 코일)는 전류를 자기 에너지 형태로 저장하는 수동 소자입니다. 도선을 코일 형태로 감은 구조로, 전류의 변화를 방해하는 특성(인덕턴스)을 가집니다.

**단위**: 헨리(H, Henry) — 실용 단위는 mH(밀리헨리), μH(마이크로헨리)  
**기호**: L  
**기본식**: V = L × (dI/dt) (전압 = 인덕턴스 × 전류 변화율)

아두이노 프로젝트에서는 DC-DC 컨버터(벅/부스트 회로), RF 필터, EMI 노이즈 제거, LC 필터 등에 사용됩니다. 저항, 커패시터에 비해 초보자에게는 덜 사용되지만, 전원 회로 설계에서 필수적입니다.

### 단위 변환

| 단위 | 표기 | 값 |
|------|------|----|
| 1 헨리 | 1 H | 1 H |
| 1 밀리헨리 | 1 mH | 10⁻³ H |
| 1 마이크로헨리 | 1 μH | 10⁻⁶ H |
| 1 나노헨리 | 1 nH | 10⁻⁹ H |

## 종류

### 1. 공심 코일 (Air Core Inductor)
- 코어(심) 없이 공기만 있는 구조
- 인덕턴스 낮음 (nH ~ 수백 nH)
- 고주파 특성 우수, 노이즈 없음
- RF 회로, 안테나 매칭에 사용

### 2. 철심 코일 (Iron Core Inductor)
- 철(Fe) 또는 규소강 코어 사용
- 높은 인덕턴스 값, 높은 포화 전류
- 전원 필터, 60Hz 라인 필터에 사용
- 무겁고 부피가 큼

### 3. 페라이트 코어 코일 (Ferrite Core Inductor)
- 페라이트(자성 세라믹) 코어 사용
- 중고주파 특성 우수, 손실 적음
- DC-DC 컨버터, EMI 필터에 많이 사용
- **아두이노/ESP32 전원 회로에서 가장 흔히 사용**

### 4. 파워 인덕터 (Power Inductor)
- DC-DC 컨버터 설계에 최적화
- 포화 전류(Saturation Current)가 높게 설계
- SMD 및 DIP 형태 모두 있음
- 4.7μH / 10μH / 22μH / 47μH / 100μH 등이 흔함

### 5. 페라이트 비드 (Ferrite Bead)
- 엄밀히는 인덕터와 다르지만 같은 범주
- 고주파 노이즈를 열로 변환하여 흡수
- USB 전원 라인, 데이터 라인 EMI 필터에 사용
- 저항처럼 생겼지만 기능이 다름

### 6. 공통 모드 초크 (Common Mode Choke)
- 두 코일을 하나의 코어에 감은 구조
- 공통 모드 노이즈(EMI) 제거에 특화
- USB, Ethernet, 모터 드라이버 라인에 사용

### 7. SMD 인덕터
- 표면 실장형, 0402 ~ 1210 크기
- 아두이노 모듈 기판에서 자주 보임

## 핀/단자 구성

### 축방향 (Axial) 인덕터

```
      ┌────────────────┐
──────┤   인덕터 코일   ├──────
 리드1  └────────────────┘  리드2
      (색상 코드 또는 값 인쇄)

극성 없음 — 어느 방향으로 연결해도 무방
```

### SMD 파워 인덕터

```
     ┌──────────┐
─────┤          ├─────
  패드1  코일 본체  패드2
     └──────────┘

마킹: 인덕턴스 값 또는 코드 표기
극성 없음
```

### 페라이트 비드 (2단자)

```
     ┌──────┐
─────┤ FB   ├─────
     └──────┘
저항처럼 생겼지만 인덕터 기능
고주파에서 저항으로 동작
```

## 동작 원리

### 에너지 저장

인덕터는 전류가 흐를 때 주변에 자기장을 형성하고, 이 자기장에 에너지를 저장합니다.

```
에너지: E = ½ × L × I²
(에너지 = ½ × 인덕턴스 × 전류²)
```

### 전류 변화 방해 (렌츠의 법칙)

- **전류 증가 시**: 역기전력(back-EMF)을 발생시켜 전류 증가를 방해
- **전류 감소 시**: 전류가 계속 흐를 수 있도록 에너지를 방출

```
전압: V = L × (ΔI / Δt)
전류 변화율이 클수록 높은 전압이 발생
```

### DC-DC 컨버터에서의 역할

**벅 컨버터 (강압형)**:
```
스위치 ON  → 인덕터에 에너지 저장 (전류 증가)
스위치 OFF → 인덕터가 에너지 방출 (전류 유지)
결과: 출력 전압을 낮추면서 전류를 평활화
```

**부스트 컨버터 (승압형)**:
```
스위치 ON  → 인덕터에 에너지 저장 (GND로 전류)
스위치 OFF → 인덕터 에너지가 출력 측으로 방출 (전압 상승)
결과: 입력보다 높은 전압 출력
```

### 임피던스 (AC에서)

```
유도성 리액턴스: XL = 2π × f × L

주파수가 높을수록 임피던스 증가
→ 고주파 신호는 차단, 저주파/DC는 통과 (저역통과 필터 특성)
```

## 아두이노 회로 연결 방법

### 1. 전원 라인 EMI 필터

```
[전원 입력] ──[인덕터 100μH]──┬── [아두이노 VCC]
                              │
                            [100μF] 전해
                              │
[GND] ────────────────────────┴── [아두이노 GND]

목적: 전원 라인의 고주파 노이즈 제거
인덕터: 정격 전류가 회로 소모 전류 이상인 것 선택
```

### 2. LC 저역통과 필터 (오디오/아날로그)

```
입력 ──[L: 1mH]──┬── 출력
                 │
                [C: 10μF]
                 │
                GND

차단 주파수: fc = 1 / (2π × √(L × C))
예: L=1mH, C=10μF → fc = 1 / (2π × √(0.001 × 0.00001)) ≈ 1.6 kHz
```

### 3. 페라이트 비드 USB 노이즈 필터

```
USB VBUS ──[FB 600Ω@100MHz]──── MCU VCC
USB D+   ──[FB 33Ω@100MHz] ──── MCU D+
USB D-   ──[FB 33Ω@100MHz] ──── MCU D-

목적: USB 라인을 통한 EMI 방출 방지 및 외부 노이즈 차단
```

### 4. 간단한 벅 컨버터 (5V → 3.3V, 참고용)

```
5V 입력 ──[스위치/MOSFET]──[L: 10μH]──── 3.3V 출력
                                   │
                               [C: 47μF]
                                   │
                                  GND
                     ↑
                 [다이오드(환류)]
                    GND 방향

실제 구현: AMS1117-3.3 리니어 레귤레이터 또는 MT3608/LM2596 IC 사용 권장
```

## 마킹/표기법 읽는 법

### 축방향 인덕터 색상 코드 (저항 코드와 유사)

4색 띠 인덕터 (단위: μH):

| 색상 | 숫자 | 승수 |
|------|------|------|
| 검정 | 0 | ×1 |
| 갈색 | 1 | ×10 |
| 빨강 | 2 | ×100 |
| 주황 | 3 | ×1k |
| 노랑 | 4 | ×10k |
| 초록 | 5 | — |
| 파랑 | 6 | — |
| 보라 | 7 | — |
| 회색 | 8 | — |
| 흰색 | 9 | — |
| 금색 | — | ×0.1 |
| 은색 | — | ×0.01 |

**예시**: 갈색-검정-갈색-은색 → 1, 0, ×10 = 100μH ±10%

### SMD 인덕터 마킹

**숫자+문자 코드** (단위: nH):
- `100` → 10 × 10⁰ = 10 nH
- `101` → 10 × 10¹ = 100 nH
- `102` → 10 × 10² = 1000 nH = 1 μH
- `103` → 10 × 10³ = 10 μH
- `4R7` 또는 `4.7` → 4.7 μH
- `R47` 또는 `.47` → 0.47 μH = 470 nH

**직접 수치 표기**:
- `10uH`, `4.7uH`, `100nH` 등 직접 인쇄

### 페라이트 비드 표기

임피던스@주파수로 표기:
- `600Ω@100MHz` → 100MHz에서 600Ω 임피던스

## 주의사항

### 1. 포화 전류 초과 금지
- 인덕터에 포화 전류 이상이 흐르면 인덕턴스가 급격히 감소
- DC-DC 컨버터에서 포화 시 효율 저하 및 과열 발생
- **설계 전류의 1.5배 이상 포화 전류를 가진 인덕터 선택**

### 2. 역기전력(Back-EMF) 주의
- 인덕터에 흐르는 전류를 갑자기 차단하면 수백 ~ 수천 V의 고전압 스파이크 발생
- 릴레이 코일, 솔레노이드 등 유도성 부하에는 반드시 **환류 다이오드(Flyback Diode)** 설치
- 아두이노 핀에서 직접 인덕터를 구동하지 말 것

```
[GPIO] ──[MOSFET/트랜지스터]──[솔레노이드 코일]──[VCC]
                    │
                  [다이오드] ← 역기전력 보호 (캐소드를 VCC 방향으로)
                    │
                   GND
```

### 3. DC 저항 (DCR) 손실
- 인덕터 내부 도선에 저항이 있어 전력 손실 발생
- 고전류 회로에서는 DCR이 낮은 인덕터 선택

### 4. 자기 간섭
- 인덕터는 자기장을 방출하여 주변 회로(마이크, 센서)에 간섭 가능
- 차폐형(Shielded) 인덕터 또는 토로이달(Toroidal) 코어 사용으로 누설 자속 감소

### 5. 극성 없음
- 일반 인덕터는 극성이 없어 어느 방향으로 연결해도 무방
- 단, 커플드 인덕터(Coupled Inductor), 공통 모드 초크는 배선 방향 중요

## 자주 쓰이는 값/사양 목록

| 인덕턴스 | 용도 | 비고 |
|---------|------|------|
| 10nH ~ 100nH | RF 매칭, 임피던스 조정 | 공심 또는 세라믹 코어 |
| 1μH ~ 4.7μH | 고전류 벅 컨버터 | 포화 전류 높은 파워 인덕터 |
| 10μH | DC-DC 컨버터 범용 | 가장 흔한 값 |
| 22μH ~ 47μH | 부스트 컨버터, 필터 | 중간 전류 |
| 100μH | 전원 필터, 초크 | 낮은 전류 회로 |
| 1mH | 오디오 저역 필터 | 낮은 차단 주파수 |
| 10mH | 낮은 주파수 필터, 측정 | 낮은 전류 |
| 페라이트 비드 | EMI 필터 (USB, 전원) | 600Ω@100MHz 타입 흔함 |

## 데이터시트 주요 파라미터 설명

| 파라미터 | 영문명 | 설명 |
|----------|--------|------|
| 인덕턴스 | Inductance (L) | 기본 인덕턴스 값, 단위 H |
| 허용 오차 | Tolerance | 인덕턴스 오차 (파워 인덕터는 ±20~30% 흔함) |
| 정격 전류 | Rated Current (Ir) | 온도 상승 기준 최대 연속 전류 (DCR 발열 기준) |
| 포화 전류 | Saturation Current (Isat) | 인덕턴스가 지정 % 감소하는 전류 (보통 20~30% 감소 기준) |
| DC 저항 | DC Resistance (DCR) | 코일 도선의 직류 저항 |
| Q 팩터 | Quality Factor (Q) | 무손실 정도, 높을수록 고효율 |
| 자기 공진 주파수 | Self Resonant Frequency (SRF) | 기생 커패시턴스와 공진하는 주파수 (이 이상에서는 인덕터가 아님) |
| 동작 주파수 | Operating Frequency | 효율적으로 동작하는 주파수 범위 |
| 코어 재질 | Core Material | 페라이트, 철분말, 공심 등 |
| 차폐 여부 | Shielded / Unshielded | 자기장 차폐 여부 |
| 온도 상승 | Temperature Rise | 정격 전류 인가 시 주위 온도 대비 상승량 |
