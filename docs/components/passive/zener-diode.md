# 제너 다이오드 (Zener Diode)

## 개요

제너 다이오드(Zener Diode)는 역방향으로 일정 전압(제너 전압, Vz)이 가해지면 전류가 흐르는 특수 다이오드입니다. 이 특성을 이용해 전압을 일정하게 유지(전압 클램핑, 전압 레퍼런스)하는 데 사용됩니다.

**기호**: ZD  
**핵심 특성**: 역방향 항복(Zener Breakdown) 전압에서 안정적으로 동작  
**주요 용도**: 전압 레퍼런스, 과전압 보호, 전압 클램핑, 간이 전압 레귤레이터

일반 다이오드는 역방향 항복 시 파괴되지만, 제너 다이오드는 적절한 전류 제한 하에서 역방향 항복 영역에서 반복적으로 동작하도록 설계되어 있습니다.

---

## 종류

### 1. 표준 제너 다이오드
- 가장 일반적인 형태
- 제너 전압: 2.4V ~ 200V 범위
- **BZX55** 시리즈: 0.5W, 소형 (DO-35 패키지)
- **1N47xx** 시리즈: 1W (DO-41 패키지)
- **1N52xx** 시리즈: 0.5W (DO-35 패키지)
- **BZX79** 시리즈: 0.5W (유럽 표준)

### 2. 정밀 제너 다이오드
- 허용 오차 ±2% 이하
- 전압 기준(Voltage Reference)용
- LM4040, TL431 등 능동 소자가 더 정밀

### 3. 저노이즈 제너 다이오드
- 오디오, 계측 회로용
- 노이즈가 적게 발생하도록 설계

### 4. 표면 실장형 (SMD)
- SOD-123, SOD-80, MELF 패키지
- **BZV55** 시리즈, **MMSZ** 시리즈
- PCB 자동화 생산에 사용

### 5. TVS 다이오드 (Transient Voltage Suppressor)
- 제너 다이오드의 고출력 버전
- 순간적인 대전류 서지(ESD, 낙뢰) 흡수
- 단방향/양방향 제품 있음
- **P6KE** 시리즈, **SMAJ** 시리즈

## 핀/단자 구성

### DO-41 / DO-35 패키지

```
   캐소드(K, -)   애노드(A, +)
        │              │
   ─────┤    ┌────┐    ├─────
        │    │ ZD │    │
        └────┴────┴────┘
        은색/회색 띠 = 캐소드(-)

회로 기호 (일반 다이오드와 다름):
     A(+)    K(-)
      │       │
      ▼ ── ─┤Z
    애노드  캐소드 (꼬리 부분에 꺾임이 있음)

제너 동작 방향:
[+전압] ─── 캐소드(K) ─ [제너 다이오드] ─ 애노드(A) ─── GND
      역방향 연결이 정상 사용 방향!
```

**정상 다이오드와 차이**:
- 외형은 일반 다이오드와 동일
- **역방향으로 연결하여 사용**하는 것이 기본 용법
- 본체에 제너 전압 또는 부품 번호 인쇄

## 동작 원리

### 제너 효과와 항복 메커니즘

**1. 제너 효과 (낮은 전압, <5V)**:
- 강한 전기장에 의해 가전자대의 전자가 직접 전도대로 이동
- 역방향 전류 급격히 증가

**2. 애벌런치 효과 (높은 전압, >6V)**:
- 역방향 전계에서 캐리어들이 충돌 이온화를 일으켜 전류 증가
- 실제 대부분의 "제너 다이오드"는 애벌런치 항복

**3. 전압 안정화 동작**:
```
전압이 Vz 이상으로 오르려 하면 → 제너가 더 많은 전류를 통과시킴
전류가 늘면 → 직렬 저항에서 전압 강하 증가
결과: 제너 양단 전압은 Vz로 유지됨

VCC ──[R]──┬── 출력 전압(= Vz)
           │
         [ZD] (역방향 연결)
           │
          GND
```

### 온도 특성

- **5V 이하 제너**: 음의 온도 계수 (온도 상승 시 전압 감소)
- **5V 이상 제너**: 양의 온도 계수 (온도 상승 시 전압 증가)
- **5.1V ~ 5.6V 제너**: 온도 계수가 거의 0에 가까움 → 전압 레퍼런스에 적합

## 아두이노 회로 연결 방법

### 1. 간이 정전압 회로 (작은 전류용)

```
[VCC 9V ~ 12V]
       │
      [R] ← 전류 제한 저항
       │
       ├──────── 출력 (5.1V)
       │
     [ZD 5.1V] ← 역방향 연결 (캐소드 위, 애노드 아래)
     (K 위, A 아래)
       │
      GND

저항 계산:
R = (VCC - Vz) / (Iz + IL)
Iz: 제너 동작 전류 (최소 5mA 이상 필요)
IL: 부하 전류

예: VCC=9V, Vz=5.1V, 부하 10mA, Iz=10mA
R = (9 - 5.1) / (0.010 + 0.010) = 3.9V / 0.020A = 195Ω → 180Ω 또는 220Ω
전력: P = (9-5.1)² / 195 ≈ 0.078W → 1/4W 저항 가능
```

### 2. GPIO 과전압 보호 (5V → ESP32 3.3V)

```
[5V 신호] ──[R: 1kΩ]──┬── ESP32 GPIO
                      │
                    [ZD 3.3V] ← 캐소드 위
                    (역방향)
                      │
                     GND

동작: 신호가 3.3V 초과 시 제너가 초과분 클램핑
주의: 더 정확한 보호는 레벨 시프터(74AHCT125 등) 사용 권장
```

### 3. 전압 레퍼런스 (ADC 기준 전압)

```
[3.3V] ──[R: 2.2kΩ]──┬── ADC AREF (ESP32에는 내장 AREF)
                     │
                   [ZD 3.3V 정밀형]
                   (역방향)
                     │
                    GND

참고: ESP32는 AREF 핀이 없어 효용 낮음
Arduino UNO (AREF 핀 있음)에서 더 유용
```

### 4. 배터리 과충전 보호 (단순)

```
[충전 전압] ──[R: 100Ω]──[충전 전류 흐름]
                  │
                [ZD 4.2V] ← 역방향
                  │
                 GND

동작: 배터리 전압이 4.2V 초과 시 제너가 과전류를 우회
경고: 정밀 배터리 관리는 전용 IC(TP4056 등) 사용 필수
제너는 보조 안전장치 개념으로만 사용
```

### 5. 클램핑 회로 (신호 진폭 제한)

```
[신호 입력] ──┬── [처리 회로]
             │
           [ZD1] ─── GND (양방향 클램핑)
           [ZD2] ─── VCC
       
또는 양방향 TVS 다이오드 1개로 대체 가능
```

## 주의사항

### 1. 반드시 역방향으로 연결
- 제너 다이오드는 역방향으로 연결해야 정상 기능
- 순방향으로 연결하면 일반 다이오드처럼 동작 (0.7V 강하)
- **캐소드(띠 있는 쪽)를 고전위(+)에 연결**

### 2. 직렬 저항 필수
- 제너만 연결하면 전류가 무한히 흐르게 됨 → 소자 파괴
- 반드시 전류 제한 저항을 직렬로 연결
- 최소 제너 전류(IZmin): 보통 5mA 이상 필요
- 최대 제너 전류(IZmax) = PZ / VZ

### 3. 전력 초과 주의
- P = VZ × IZ (제너 전력 소모)
- 1W 제너 사용 시 최대 전류: IZmax = 1W / VZ
- 예: 5.1V 제너 → IZmax = 1 / 5.1 ≈ 196mA

### 4. 온도 특성 고려
- 정밀 회로에서는 온도 계수가 0에 가까운 5.1V ~ 5.6V 제품 선택
- 온도 보상이 필요하면 정밀 전압 기준 IC(LM4040, TL431) 사용 권장

### 5. 노이즈 발생
- 제너 다이오드는 동작 시 화이트 노이즈를 발생시킴
- 정밀 아날로그 회로에서는 문제가 될 수 있음
- 대안: TL431 정밀 전압 레귤레이터

### 6. TVS와 혼동 주의
- TVS 다이오드는 외형이 유사하지만 서지 보호용
- 둘 다 비슷한 기호를 사용하므로 데이터시트 확인 필수

## 데이터시트 주요 파라미터 설명

| 파라미터 | 영문명 | 설명 |
|----------|--------|------|
| 제너 전압 | Zener Voltage (VZ) | 지정 전류(IZT)에서의 역방향 항복 전압 |
| 테스트 전류 | Test Current (IZT) | VZ 측정 기준 전류 (보통 최대 전류의 20~25%) |
| 최소 제너 전류 | Minimum Zener Current (IZK) | 안정적인 동작을 위한 최소 전류 |
| 최대 제너 전류 | Maximum Zener Current (IZM) | 허용되는 최대 역방향 전류 |
| 최대 전력 소모 | Power Dissipation (PZ) | 최대 허용 전력 (W) |
| 동적 임피던스 | Zener Impedance (ZZT) | 지정 전류에서의 AC 임피던스 (낮을수록 안정) |
| 전압 허용 오차 | Voltage Tolerance | VZ의 허용 오차 범위 (A등급: ±5%, B등급: ±2%) |
| 온도 계수 | Temperature Coefficient (αVZ) | 온도에 따른 VZ 변화율 (mV/°C 또는 %/°C) |
| 접합 온도 | Junction Temperature (TJ) | 최대 허용 접합 온도 |
| 역방향 누설 전류 | Reverse Current (IR) | 역방향 전압 인가 시 항복 이전의 누설 전류 |
| 순방향 전압 | Forward Voltage (VF) | 순방향 도통 시 전압 강하 (~0.7V) |
