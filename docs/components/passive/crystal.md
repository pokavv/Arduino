# 크리스탈 발진자 (Crystal Oscillator)

## 개요

크리스탈 발진자(Crystal Oscillator, 수정 발진자)는 수정(석영, Quartz) 결정체의 **압전 효과(Piezoelectric Effect)**를 이용하여 매우 정밀하고 안정적인 주파수를 발생시키는 소자입니다.

**압전 효과**: 수정에 전압을 가하면 기계적으로 진동하고, 반대로 기계적 진동을 가하면 전기 신호를 발생시키는 현상

**주요 역할**:
- 마이크로컨트롤러(MCU)의 클록 신호 생성
- UART, SPI, I2C 등 통신 타이밍 기준
- RTC(실시간 클록) 타이밍 기준

아두이노 프로젝트에서 크리스탈은 주로 **정확한 통신 속도**(보레이트 오류 방지)와 **정확한 시간 측정**이 필요할 때 사용됩니다. ESP32-C3는 내장 RC 오실레이터 외에 외부 크리스탈을 지원합니다.

## 종류

### 1. 기본 크리스탈 (Crystal, XTAL)
- 외부 발진 회로(인버터 + 부하 커패시터)가 필요
- 가장 저렴하고 일반적
- **HC-49S** 패키지: 가장 흔한 관통홀 형태
- **HC-49U**: 높이가 더 낮은 슬림형
- 주파수 범위: 수십 kHz ~ 수십 MHz

### 2. 크리스탈 오실레이터 (Crystal Oscillator, OSC)
- 발진 회로가 내장된 완성 모듈
- 4핀 패키지 (VCC, GND, OUTPUT, NC/OE)
- 별도 외부 부품 없이 바로 클록 신호 출력
- 더 정밀하고 안정적이지만 더 비쌈
- DIP-4, SMD 패키지

### 3. 음차 크리스탈 (Tuning Fork Crystal)
- 시계 모양(음차, Tuning Fork)으로 만들어진 32.768kHz 크리스탈
- **32.768 kHz** = 2¹⁵ Hz → 15번 분주하면 1Hz = 1초
- RTC(실시간 클록) 전용
- 원통형 소형 패키지 (지름 약 2mm, 길이 약 6mm)

### 4. TCXO (온도 보상 크리스탈 오실레이터)
- 온도 변화에 따른 주파수 편차를 보상하는 회로 내장
- GPS, 통신 장비에 사용
- 일반 크리스탈 대비 10~100배 정밀도

### 5. OCXO (온도 조절 크리스탈 오실레이터)
- 히터로 크리스탈을 일정 온도로 유지
- 매우 높은 정밀도, 고가, 소비전력 큼
- 계측기, 통신 기지국 등

### 6. SMD 크리스탈
- 표면 실장형, 2핀 또는 4핀
- 3225 (3.2×2.5mm), 2016 (2.0×1.6mm) 등 크기 규격
- 아두이노 호환 모듈 기판에 자주 사용

## 핀/단자 구성

### HC-49S 패키지 (관통홀, 2핀)

```
      ┌──────────┐
      │          │
      │ CRYSTAL  │
      │          │
      └────┬─┬───┘
           │ │
        핀1  핀2
        (XIN) (XOUT)
        
극성 없음 — 어느 방향으로 연결해도 무방
```

### 4핀 크리스탈 오실레이터 (DIP-4)

```
    핀1(NC/OE)   핀2(GND)
         │            │
    ┌────┴────────────┴────┐
    │                      │
    │   크리스탈 오실레이터    │
    │                      │
    └────┬────────────┬────┘
         │            │
    핀4(OUT)     핀3(VCC)

핀 배치:
1번 핀: NC (No Connect) 또는 OE (Output Enable)
2번 핀: GND
3번 핀: VCC (+3.3V 또는 +5V)
4번 핀: CLK OUT (클록 출력)

점 또는 홈이 있는 쪽이 1번 핀
```

### 음차 크리스탈 (32.768 kHz, 원통형)

```
       ┌──────────┐
       │  32.768k │
       │   XTAL   │
       └─┬──────┬─┘
         │      │
        핀1    핀2
       극성 없음
```

## 동작 원리

### 압전 효과 (Piezoelectric Effect)

```
전기 → 기계 (역압전 효과):
전압 인가 → 수정 결정 변형 → 기계적 진동

기계 → 전기 (압전 효과):
기계적 진동 → 전기 신호 발생
```

### 발진 회로에서의 동작

크리스탈 단독으로는 발진하지 않고, 외부 발진 회로(인버터 또는 CMOS NOT 게이트)와 함께 동작합니다.

```
발진 회로 기본 구성:
                Rf (피드백 저항, 1MΩ)
          ┌────────────────────┐
          │                   │
MCU XTAL1 ─┤ NOT 게이트/인버터 ├─ MCU XTAL2
(XIN)      │                   │   (XOUT)
          └───────────────────┘
               │           │
             [C1]         [C2]  ← 부하 커패시터
               │           │
              GND         GND
```

**동작 순서**:
1. 전원 인가 시 회로에서 잡음 발생
2. 크리스탈의 공진 주파수 성분만 선택적으로 증폭
3. 양의 피드백으로 발진 지속
4. 크리스탈의 기계적 공진이 주파수를 정밀하게 고정

### 직렬 공진과 병렬 공진

```
직렬 공진 주파수 (fs): 임피던스 최소
병렬 공진 주파수 (fp): 임피던스 최대

실제 사용: 부하 커패시터(CL)에 의해 fs < 동작 주파수 < fp 범위
부하 커패시터 값이 크면 주파수 낮아짐
```

## 아두이노 회로 연결 방법

### 1. Arduino (ATmega328P) 외부 크리스탈 연결

```
MCU XTAL1(핀9) ──┬── 크리스탈 핀1
MCU XTAL2(핀10)──┤── 크리스탈 핀2
                 │
               [C1: 22pF] ── GND  (XTAL1 쪽)
               [C2: 22pF] ── GND  (XTAL2 쪽)

크리스탈: 16MHz (Arduino UNO 표준)
부하 커패시터: 22pF (HC-49S 기준)
주의: 배선은 최대한 짧게 (노이즈 민감)
```

### 2. ESP32-C3 외부 크리스탈 연결

```
ESP32-C3 XTAL_P ── 크리스탈 핀1
ESP32-C3 XTAL_N ── 크리스탈 핀2
                   │           │
                 [C: 10pF]   [C: 10pF]
                   │           │
                  GND         GND

권장 크리스탈: 40MHz 또는 26MHz (ESP32-C3 지원 주파수 확인)
부하 커패시터: 10pF (Espressif 권장, 보드 기생 커패시턴스 포함)
참고: ESP32-C3 Super Mini는 이미 기판에 크리스탈 탑재됨
```

### 3. RTC 모듈 (DS3231, DS1307) 32.768 kHz 크리스탈

```
RTC IC X1 ── 32.768kHz 크리스탈 핀1
RTC IC X2 ── 32.768kHz 크리스탈 핀2
             │                │
           [C: 6~12.5pF]    [C: 6~12.5pF]
             │                │
            GND              GND

참고: DS3231 모듈은 온도 보상 오실레이터 내장 (외부 크리스탈 불필요)
DS1307은 외부 32.768kHz 크리스탈 필요
```

### 4. 크리스탈 오실레이터 모듈 사용 (더 간단)

```
크리스탈 오실레이터 모듈 (4핀 DIP):
VCC(3번) ── 3.3V 또는 5V
GND(2번) ── GND
OUT(4번) ── MCU CLK 입력
1번(NC)  ── 연결 안함

장점: 외부 부하 커패시터 불필요, 바로 클록 신호 출력
용도: 정밀도 요구 높은 경우 또는 레이아웃 공간 부족 시
```

## 마킹/표기법 읽는 법

### HC-49S 패키지 마킹

```
본체에 주파수 직접 표기:
- "16.000" — 16.000 MHz
- "8.000" — 8.000 MHz
- "32.768K" — 32.768 kHz
- "11.0592" — 11.0592 MHz

제조사 로고, 로트 번호 등 추가 표기
```

### SMD 크리스탈 마킹

크기가 작아 마킹이 없거나 간략함:
- 숫자만 표기: "160" → 16.0 MHz
- 포장 라벨로 구분

### 크리스탈 오실레이터 (4핀 패키지)

```
- 주파수 직접 표기: "16.000 MHz", "25.000 MHz"
- 전압 표기: "3.3V" 또는 "5V"
- 핀 1번 위치: 패키지 모서리의 점(dot) 또는 홈(notch)
```

## 주의사항

### 1. 부하 커패시터 값 맞추기
- 크리스탈마다 지정된 부하 커패시턴스(CL)가 있음
- 잘못된 CL 값 → 주파수 오차 발생
- 일반적으로 HC-49S 16MHz → 22pF, 32.768kHz → 12.5pF
- 기판의 기생 커패시턴스(3~5pF)를 고려하여 계산:
  `CL = (C1 × C2) / (C1 + C2) + 기생 커패시턴스`

### 2. 배선 최대한 짧게
- 크리스탈 회로는 고주파 신호이므로 배선이 길면 노이즈 발생
- MCU의 XTAL 핀에서 크리스탈까지 5mm 이내 권장
- 크리스탈 아래에 접지 패턴을 두어 차폐 효과

### 3. 드라이브 레벨 초과 금지
- 너무 강한 신호를 가하면 크리스탈 과구동(Over-drive) 발생
- 크리스탈이 분열되거나 주파수 불안정
- MCU 설계 시 드라이브 레벨 설정 확인 (ESP32는 자동)

### 4. 주파수 선택 (UART 보레이트)
- 정확한 UART 통신을 위해 보레이트 오차를 최소화하는 크리스탈 선택
- 11.0592 MHz: 모든 표준 보레이트에서 오차 0%
- 16 MHz: 대부분 보레이트에서 오차 0.2% 이내 (허용 범위)
- 8 MHz (Arduino Pro Mini 3.3V 버전): 일부 보레이트 오차 큼

### 5. 온도 영향
- 크리스탈 주파수는 온도에 따라 변동
- 표준 크리스탈: ±50 ppm/°C 수준
- 정밀 타이밍이 필요하면 TCXO 사용

### 6. 시작 시간 (Start-up Time)
- 전원 인가 후 발진이 안정화될 때까지 수ms ~ 수백ms 소요
- MCU 리셋 후 발진 안정 전까지 클록이 불안정할 수 있음
- MCU 설계에서 크리스탈 안정화 대기 시간 고려

### 7. 기생 커패시턴스 영향
- PCB 배선, MCU 핀 등의 기생 커패시턴스가 부하에 추가됨
- 실제 부하 커패시터 값은 지정값보다 약간 작게 선택

## 데이터시트 주요 파라미터 설명

| 파라미터 | 영문명 | 설명 |
|----------|--------|------|
| 공칭 주파수 | Nominal Frequency (f₀) | 설계 기준 주파수 |
| 주파수 허용 오차 | Frequency Tolerance | 25°C에서 공칭 주파수 대비 오차 (ppm) |
| 주파수 안정성 | Frequency Stability | 동작 온도 범위 내 최대 주파수 변동 (ppm) |
| 부하 커패시턴스 | Load Capacitance (CL) | 발진 회로에 연결해야 하는 외부 커패시턴스 |
| 등가 직렬 저항 | Equivalent Series Resistance (ESR / R1) | 크리스탈의 내부 저항, 낮을수록 발진 용이 |
| 션트 커패시턴스 | Shunt Capacitance (C0) | 전극 간 기생 커패시턴스 |
| 동적 커패시턴스 | Motional Capacitance (C1) | 기계적 진동과 등가 커패시턴스 |
| 드라이브 레벨 | Drive Level (DL) | 크리스탈에 가해지는 최대 전력 |
| 노화 | Aging | 시간이 지남에 따른 주파수 변화율 (ppm/년) |
| 직렬 공진 주파수 | Series Resonant Frequency (fs) | 직렬 임피던스 최소가 되는 주파수 |
| 병렬 공진 주파수 | Parallel Resonant Frequency (fp) | 병렬 임피던스 최대가 되는 주파수 |
| Q 팩터 | Quality Factor (Q) | 공진의 날카로움, 높을수록 안정적 (수정: 10,000~500,000) |
| 삽입 손실 | Insertion Loss | 직렬 공진에서의 신호 손실 (dB) |
