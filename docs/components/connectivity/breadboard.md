# 브레드보드 (Breadboard)

## 개요

브레드보드(Solderless Breadboard)는 납땜 없이 전자 부품과 전선을 꽂아 회로를 구성할 수 있는 프로토타입 제작 도구입니다.
이름의 유래는 초기 라디오 제작자들이 실제 빵 자르는 판 위에 못을 박아 회로를 구성하던 것에서 비롯되었습니다.

Arduino 프로젝트에서는 주로 다음 용도로 사용합니다.
- 설계 중인 회로를 납땜 없이 빠르게 프로토타입
- 저항, 캐패시터, IC 등 부품 임시 연결
- 여러 가지 연결 방법을 실험하고 비교
- 학습 및 교육 목적

## 내부 구조 및 연결 패턴

### 풀 사이즈(830홀) 브레드보드 레이아웃

```
  +  -                                        +  -
  [ 전원 레일 1 ]                              [ 전원 레일 2 ]
  (A-E 구역 5홀 × 30행 × 양쪽 갭 존재)
  a  b  c  d  e     f  g  h  i  j
1 [ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
2 [ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
3 [ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
...
30[ ][ ][ ][ ][ ]   [ ][ ][ ][ ][ ]
```

### 연결 방향 규칙

| 구역 | 연결 방향 | 설명 |
|------|----------|------|
| 전원 레일 (+, −) | **수평** | 한 레일의 모든 홀이 연결됨 (단, 중간에 끊김 있음) |
| 중앙 본체 구역 | **수직** | 같은 행(Row)의 a-b-c-d-e가 연결, f-g-h-i-j가 별도 연결 |

```
전원 레일 상세:
  + − + − + − + − (총 25홀 × 2줄 = 50홀 기준)
  ← 왼쪽 절반 모두 연결 ─ 오른쪽 절반 모두 연결 →
  주의: 풀 사이즈에서 가운데 전원 레일이 끊겨있는 제품이 많습니다!
  → 전원 레일 연속성을 멀티미터로 반드시 확인

본체 구역:
  행 1: a1-b1-c1-d1-e1 → 5홀 서로 연결 (수직 방향 스트립)
         f1-g1-h1-i1-j1 → 5홀 서로 연결 (수직 방향 스트립)
         (a1-e1 그룹 ↔ f1-j1 그룹은 중앙 갭으로 분리됨)
```

## 동작 원리

브레드보드 내부는 금속 클립(스프링 접촉자)이 수직 또는 수평으로 배열되어 있습니다.

**본체 수직 스트립**:
```
[클립 1개] 5개 홀을 아래로 연결
핀을 삽입하면 클립의 스프링 압력으로 전기 접촉
```

**전원 레일 수평 스트립**:
```
[긴 클립 1개] 25~50개 홀을 옆으로 연결
+ 레일과 − 레일은 서로 절연
```

전류 흐름:
```
핀 삽입 → 클립과 마찰 접촉 → 같은 스트립의 모든 핀에 전기적 연결
```

## 아두이노 연결 방법

### 기본 LED 점등 회로

```
Arduino 5V ────── 브레드보드 전원 레일 +
Arduino GND ───── 브레드보드 전원 레일 −

브레드보드 전원 레일 + ────── LED 양극(긴 다리, a1열)
LED 음극(짧은 다리, a2열 점프 후 저항)
저항 220Ω ─── 다른 행 (a4열)
저항 다른 쪽 (a6열) ─── 전원 레일 −
```

### IC 삽입 위치

```
DIP IC는 중앙 갭에 걸쳐 삽입:
  A열(a~e): IC 왼쪽 핀
  B열(f~j): IC 오른쪽 핀
  갭으로 분리되어 양쪽 핀이 서로 단락되지 않음
```

### 전원 분배

```
Arduino 5V → 점퍼 → 브레드보드 + 레일 (빨간 선)
Arduino GND → 점퍼 → 브레드보드 − 레일 (검은 선)
모든 부품의 VCC → 해당 행에서 + 레일로 점퍼
모든 부품의 GND → 해당 행에서 − 레일로 점퍼
```

## 주의사항

1. **전원 레일 단절 확인**: 풀 사이즈 브레드보드 전원 레일은 가운데가 끊겨있는 경우가 많습니다. 반드시 멀티미터로 연속성을 확인하고, 필요 시 점퍼 와이어로 연결하세요.
2. **핀 직경 제한**: 삽입 가능한 최대 핀 직경은 약 0.8mm. 두꺼운 점퍼 와이어(20AWG 이상)는 삽입하기 어렵거나 홀을 손상시킵니다.
3. **클립 손상**: 굵은 핀을 억지로 삽입하거나 비틀면 내부 클립이 영구 변형됩니다. 접촉 불량의 가장 큰 원인.
4. **접촉 불량**: 오래된 브레드보드는 산화로 인해 접촉 저항이 높아집니다. 원인 불명의 오동작 시 다른 브레드보드로 교체하여 확인.
5. **고전류 금지**: 모터 직결 등 1A 이상의 전류는 내부 클립이 과열될 수 있습니다. 고전류 배선은 별도 전선으로 처리.
6. **AC 전원 절대 금지**: 브레드보드는 DC 저전압(5V, 12V 등)용입니다. 220V AC를 연결하면 아크 및 화재 위험.
7. **수평 연결 착각**: 초보자가 가장 많이 실수하는 것은 "본체 구역이 수평으로 연결된다"는 착각입니다. 수평이 아닌 수직(같은 행 5홀)임을 기억하세요.

## 실사용 팁 및 회로 패턴

### 전원 레일 연속성 패턴

```
+────────────────── 끊김! ──────────────────+
−────────────────── 끊김! ──────────────────−

해결 방법:
왼쪽 + 레일 오른쪽 끝 ─── 점퍼 ─── 오른쪽 + 레일 왼쪽 끝
왼쪽 − 레일 오른쪽 끝 ─── 점퍼 ─── 오른쪽 − 레일 왼쪽 끝
```

### 색상으로 배선 정리

```
빨간 와이어: 전원(5V, 3.3V, VCC)
검은 와이어: GND
노란 와이어: 신호선 1
파란 와이어: 신호선 2
초록 와이어: 신호선 3
```

### 여러 브레드보드 연결

슬라이딩 연결 홈이 있는 브레드보드는 옆으로 붙여 연결 가능합니다.
전원 레일은 별도로 점퍼로 연결해야 합니다.

### 브레드보드 수명 연장

```
사용 후:
1. 모든 부품과 와이어 제거
2. 전용 브레드보드 솔(Brush)로 홀 청소
3. 이소프로필 알코올 + 면봉으로 표면 청소
4. 완전 건조 후 보관 (습기 차단)
```

### 회로 검증 순서

```cpp
// 브레드보드 회로 디버깅 순서
// 1. 전원 공급 확인
//    멀티미터로 + 레일 = 5V, − 레일 = 0V 확인

// 2. IC/모듈 전원 핀 확인
//    VCC 핀 전압 측정

// 3. 신호 경로 추적
//    각 연결점에서 기대 전압 측정

// 4. 접촉 불량 의심 시
//    해당 와이어를 다른 홀로 옮겨 재시험
```
