# 연산자

## 산술 연산자

```cpp
int a = 10, b = 3;

a + b   // 13  더하기
a - b   // 7   빼기
a * b   // 30  곱하기
a / b   // 3   나누기 (정수끼리면 몫만 반환, 소수점 버림)
a % b   // 1   나머지 (모듈로)
```

### 나누기 주의사항

```cpp
int a = 7, b = 2;
int 결과1 = a / b;          // 3  — 정수 나누기, 소수점 버림
float 결과2 = a / b;        // 3.0 — 나눗셈 먼저 하고 대입, 여전히 손실
float 결과3 = (float)a / b; // 3.5 — 올바른 방법: 하나를 float로 캐스팅
```

### 나머지(%) 활용

```cpp
// 짝수/홀수 판별
if (숫자 % 2 == 0) { /* 짝수 */ }

// N개마다 반복
if (카운터 % 10 == 0) { /* 10번마다 실행 */ }

// 숫자 자리수 분리
int 일의자리 = 숫자 % 10;
int 십의자리 = (숫자 / 10) % 10;
```

## 대입 연산자

```cpp
int x = 10;

x += 5;   // x = x + 5  → x = 15
x -= 3;   // x = x - 3  → x = 12
x *= 2;   // x = x * 2  → x = 24
x /= 4;   // x = x / 4  → x = 6
x %= 4;   // x = x % 4  → x = 2
```

## 증감 연산자

```cpp
int x = 5;

x++;    // x = 6  (사용 후 증가)
x--;    // x = 5  (사용 후 감소)
++x;    // x = 6  (증가 후 사용)
--x;    // x = 5  (감소 후 사용)
```

### 전위 vs 후위 차이

```cpp
int a = 5;
int b = a++;   // b = 5, a = 6  (b에 먼저 대입 후 a 증가)
int c = ++a;   // a = 7, c = 7  (a 먼저 증가 후 c에 대입)
```

단순히 값을 1 올리거나 내릴 때는 `x++`, `x--` 사용하면 됩니다.

## 비교 연산자

결과는 항상 `true` 또는 `false`입니다.

```cpp
int a = 10, b = 5;

a == b   // false  같다
a != b   // true   다르다
a > b    // true   크다
a < b    // false  작다
a >= b   // true   크거나 같다
a <= b   // false  작거나 같다
```

### 자주 하는 실수: = 와 == 혼동

```cpp
if (x = 5)  { }   // ⚠️ 대입! x에 5를 넣고 조건은 항상 true
if (x == 5) { }   // ✅ 비교! x가 5인지 확인
```

## 논리 연산자

여러 조건을 조합할 때 씁니다.

```cpp
bool a = true, b = false;

a && b   // false  AND: 둘 다 true여야 true
a || b   // true   OR: 하나만 true여도 true
!a       // false  NOT: true↔false 반전
```

### 실제 사용 예

```cpp
// 버튼 눌리고 + 온도가 30도 이상일 때
if (digitalRead(버튼) == LOW && 온도 >= 30) { }

// 버튼1 또는 버튼2 중 하나라도 눌리면
if (digitalRead(버튼1) == LOW || digitalRead(버튼2) == LOW) { }

// 연결되지 않은 경우
if (!WiFi.isConnected()) { }
```

## 비트 연산자

비트(0과 1) 단위로 연산합니다. 주로 레지스터 제어, 플래그 관리에 씁니다.

```cpp
byte a = 0b11001010;   // 2진수 표기
byte b = 0b10110101;

a & b    // AND: 두 비트 모두 1이면 1
a | b    // OR: 하나라도 1이면 1
a ^ b    // XOR: 서로 다르면 1
~a       // NOT: 모든 비트 반전
a << 2   // 왼쪽으로 2비트 시프트 (× 4)
a >> 1   // 오른쪽으로 1비트 시프트 (÷ 2)
```

### 비트 연산 실용 예

```cpp
byte 상태 = 0b00000000;

// 특정 비트 켜기 (OR 사용)
상태 |= (1 << 3);   // 3번 비트를 1로

// 특정 비트 끄기 (AND + NOT 사용)
상태 &= ~(1 << 3);  // 3번 비트를 0으로

// 특정 비트 확인
if (상태 & (1 << 3)) { /* 3번 비트가 1이면 */ }

// 특정 비트 토글 (XOR 사용)
상태 ^= (1 << 3);   // 3번 비트 반전
```

## 삼항 연산자

`조건 ? 참일때값 : 거짓일때값`

if/else를 한 줄로 줄일 때 씁니다.

```cpp
int 절댓값 = (x >= 0) ? x : -x;

String 상태 = (LED켜짐) ? "ON" : "OFF";

digitalWrite(LED_PIN, (켜짐) ? LOW : HIGH);
```

## 연산자 우선순위

높은 순서대로:

```
1. ()          괄호
2. !, ~, ++, --  단항 연산자
3. *, /, %     곱셈/나눗셈
4. +, -        덧셈/뺄셈
5. <<, >>      비트 시프트
6. <, <=, >, >= 비교
7. ==, !=      등호 비교
8. &           비트 AND
9. ^           비트 XOR
10. |          비트 OR
11. &&         논리 AND
12. ||         논리 OR
13. =, +=, ... 대입
```

> 헷갈리면 그냥 **괄호**를 쓰세요. 가독성도 올라갑니다.

```cpp
// 헷갈림
int 결과 = a + b * c - d / e;

// 명확함
int 결과 = a + (b * c) - (d / e);
```
