# 배열과 문자열

## 배열이 뭔가요?

같은 자료형의 값을 여러 개 연속으로 저장하는 공간입니다.
변수를 여러 개 만드는 대신 하나의 이름으로 묶어서 관리합니다.

```cpp
// 변수 5개 따로 선언 — 불편함
int 센서0 = 0;
int 센서1 = 0;
int 센서2 = 0;
int 센서3 = 0;
int 센서4 = 0;

// 배열로 묶기 — 훨씬 편함
int 센서값[5] = {0, 0, 0, 0, 0};
```

---

## 배열 선언과 접근

```cpp
// 선언과 초기화
int 숫자[5] = {10, 20, 30, 40, 50};

// 인덱스는 0부터 시작!
숫자[0]   // 10  (첫 번째)
숫자[1]   // 20  (두 번째)
숫자[4]   // 50  (다섯 번째, 마지막)
숫자[5]   // ⚠️ 범위 초과! 오류 또는 쓰레기값

// 크기만 지정 (초기값 없음 — 쓰레기값 주의)
int 버퍼[10];

// 크기 생략 (자동으로 3으로 설정)
int 핀번호[] = {3, 5, 9};
```

---

## 배열 크기 구하기

```cpp
int 핀목록[] = {3, 5, 9, 10};

// sizeof(배열) / sizeof(첫번째요소) = 원소 개수
int 개수 = sizeof(핀목록) / sizeof(핀목록[0]);  // 4
```

---

## 배열과 for 반복문

```cpp
int LED핀들[] = {3, 5, 6, 9};
int 개수 = sizeof(LED핀들) / sizeof(LED핀들[0]);

void setup() {
    for (int i = 0; i < 개수; i++) {
        pinMode(LED핀들[i], OUTPUT);
    }
}

void loop() {
    // 순서대로 켜기
    for (int i = 0; i < 개수; i++) {
        digitalWrite(LED핀들[i], HIGH);
        delay(200);
    }
    // 순서대로 끄기
    for (int i = 0; i < 개수; i++) {
        digitalWrite(LED핀들[i], LOW);
        delay(200);
    }
}
```

## 2차원 배열

배열 안에 배열 — 표(행, 열) 구조입니다.

```cpp
// 3행 2열 배열
int 표[3][2] = {
    {1, 2},
    {3, 4},
    {5, 6}
};

표[0][0]  // 1  (0행 0열)
표[1][1]  // 4  (1행 1열)
표[2][0]  // 5  (2행 0열)
```

### 실용 예: 버튼 매트릭스 핀 배치

```cpp
byte 키패드[4][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9},
    {'*', 0, '#'}
};
```

## 문자열

### String 클래스 (아두이노 편의 기능)

사용하기 편리하지만 메모리를 많이 씁니다. 소형 보드(우노)에서 많이 쓰면 불안정해집니다.

```cpp
String 이름 = "ESP32-C3";
String 메시지 = "온도: " + String(25.3) + "°C";

// 자주 쓰는 메서드
이름.length()          // 문자열 길이: 8
이름.indexOf("32")     // 부분 문자열 위치: 2
이름.substring(0, 5)   // 부분 추출: "ESP32"
이름.toUpperCase()     // 대문자: "ESP32-C3"
이름.toLowerCase()     // 소문자: "esp32-c3"
이름.replace("-", "_") // 치환: "ESP32_C3"
이름.startsWith("ESP") // true
이름.endsWith("C3")    // true
이름.trim()            // 앞뒤 공백 제거
이름.toInt()           // 숫자 문자열을 int로 변환
이름.toFloat()         // 숫자 문자열을 float로 변환

// 비교 (== 연산자 사용 가능)
if (이름 == "ESP32-C3") { }
if (이름 != "Arduino") { }
```

### 문자열 합치기

```cpp
float 온도 = 25.3;
String 출력 = "온도: " + String(온도) + "°C, 습도: " + String(60) + "%";
Serial.println(출력);  // 온도: 25.3°C, 습도: 60%
```

## char 배열 (C 스타일 문자열)

메모리가 부족한 환경에서 String 대신 사용합니다.
끝에 항상 `\0` (널 문자)이 있어야 합니다.

```cpp
char 이름[] = "ESP32";       // 자동으로 \0 추가됨 → 6바이트
char 버퍼[50];               // 50바이트 버퍼

// 주요 함수 (string.h)
strlen(이름)                 // 길이: 5
strcpy(버퍼, "Hello");       // 복사
strcat(버퍼, " World");      // 이어붙이기
strcmp(이름, "ESP32")        // 비교: 같으면 0
sprintf(버퍼, "온도: %.1f", 25.3);  // 형식 문자열로 만들기
```

### sprintf 활용

```cpp
char 버퍼[100];
float 온도 = 25.3;
int 습도 = 60;

sprintf(버퍼, "온도: %.1f°C, 습도: %d%%", 온도, 습도);
Serial.println(버퍼);
// 출력: 온도: 25.3°C, 습도: 60%
```

| 형식 지정자 | 의미 |
|------------|------|
| `%d` | 정수 |
| `%f` | 실수 |
| `%.2f` | 소수점 2자리 실수 |
| `%s` | 문자열 |
| `%c` | 문자 |
| `%%` | % 문자 자체 |

## String vs char[] 언제 쓸까?

| 상황 | 추천 |
|------|------|
| ESP32 (메모리 여유로움) | `String` |
| 아두이노 우노 (메모리 2KB) | `char[]` |
| MQTT 토픽/페이로드 | `char[]` (라이브러리 요구사항) |
| 간단한 출력용 | `String` |
