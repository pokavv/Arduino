# 저항 (Resistor)

## 개요

저항(Resistor)은 전류의 흐름을 방해하는 수동 소자입니다. 전기 회로에서 가장 기본적이고 필수적인 부품으로, 전류를 제한하거나 전압을 분배하거나 신호를 감쇠하는 데 사용됩니다.

**단위**: 옴(Ω, Ohm)  
**기호**: R  
**법칙**: 옴의 법칙 — V = I × R (전압 = 전류 × 저항)

아두이노 프로젝트에서는 LED 전류 제한, 풀업(Pull-up)/풀다운(Pull-down) 설정, 분압기(Voltage Divider), 센서 바이어싱 등에 광범위하게 사용됩니다.

---

## 종류

### 1. 탄소 피막 저항 (Carbon Film Resistor)
- 가장 일반적인 저항, 저렴하고 구하기 쉬움
- 허용 오차: ±5% (금색 띠)
- 노이즈가 약간 있어 고정밀 회로에는 부적합
- 아두이노 입문 프로젝트에서 가장 많이 사용

### 2. 금속 피막 저항 (Metal Film Resistor)
- 허용 오차: ±1% (갈색 띠)
- 온도 안정성이 좋고 노이즈가 적음
- 정밀 측정, 오디오, 계측 회로에 적합

### 3. 탄소 합성 저항 (Carbon Composition Resistor)
- 과거에 많이 사용, 현재는 드묾
- 서지(Surge) 내성이 강해 전원 회로에 사용하기도 함

### 4. 권선 저항 (Wire Wound Resistor)
- 저항값이 매우 정밀, 전력 소모가 큰 회로에 사용
- 코일 구조로 인해 인덕턴스 성분이 있어 고주파에는 부적합
- 5W, 10W 등 고전력용

### 5. SMD 저항 (Surface Mount Device)
- 표면 실장용, 소형화된 회로 기판에 사용
- 0402, 0603, 0805, 1206 등 크기 규격
- 마킹 코드로 저항값 표기 (예: 103 = 10kΩ)

### 6. 가변 저항 (Variable Resistor / Potentiometer)
- 저항값을 수동으로 조절 가능
- 트리머(Trimmer): 반고정형, 회로 내 미세 조정용
- 포텐시오미터: 볼륨 조절 등 사용자 인터페이스용

### 7. 특수 저항
- **서미스터(Thermistor)**: 온도에 따라 저항값이 변화 (NTC/PTC)
- **LDR(광 의존 저항)**: 빛의 세기에 따라 저항값이 변화
- **VDR(전압 의존 저항)**: 전압에 따라 저항값이 변화 (과전압 보호)

## 핀/단자 구성

저항은 **극성이 없는** 2단자 소자입니다.

```
    ┌───────────────────┐
────┤                   ├────
 단자1   저항 본체      단자2
(리드1)              (리드2)
```

- 어느 방향으로 연결해도 동일하게 동작합니다.
- 리드(Lead)는 두 개이며, 길이는 약 25mm (일반 1/4W 기준)
- 리드 간격(Pitch): 약 10mm (표준 폼 팩터 기준)

## 동작 원리

저항은 **전자의 흐름을 방해하는 물질**로 만들어집니다.

1. **에너지 소산**: 저항을 통과한 전자는 운동 에너지를 열로 변환합니다.
2. **전압 강하**: V = I × R 법칙에 따라 저항 양단에 전압 차이가 발생합니다.
3. **전류 제한**: 회로에 흐르는 전류를 I = V / R 으로 제한합니다.
4. **전력 소모**: P = V × I = I² × R = V² / R (단위: W)

### 저항 연결 방식

**직렬 연결 (Series)**
```
─[R1]─[R2]─[R3]─
합성 저항: R = R1 + R2 + R3
```

**병렬 연결 (Parallel)**
```
    ┌─[R1]─┐
────┤─[R2]─├────
    └─[R3]─┘
합성 저항: 1/R = 1/R1 + 1/R2 + 1/R3
두 개 병렬: R = (R1 × R2) / (R1 + R2)
```

## 아두이노 회로 연결 방법

### 1. LED 전류 제한 저항

```
ESP32 GPIO (3.3V) ─── [220Ω] ─── [LED+] ─── [LED-] ─── GND

계산 방법:
- 공급 전압: 3.3V
- LED 순방향 전압(Vf): 약 2.0V (빨간색), 3.0V (파란색/흰색)
- 원하는 전류: 10mA
- R = (Vcc - Vf) / I = (3.3 - 2.0) / 0.010 = 130Ω → 150Ω 또는 220Ω 사용
```

### 2. 풀업 저항 (버튼)

```
3.3V ─── [10kΩ] ─── GPIO 핀 ─── [버튼] ─── GND

동작:
- 버튼 미입력: GPIO = HIGH (10kΩ을 통해 3.3V)
- 버튼 입력: GPIO = LOW (GND로 직결)
참고: ESP32 내장 풀업 사용 시 외부 저항 불필요 (INPUT_PULLUP)
```

### 3. 풀다운 저항 (버튼)

```
GND ─── [10kΩ] ─── GPIO 핀 ─── [버튼] ─── 3.3V

동작:
- 버튼 미입력: GPIO = LOW (10kΩ을 통해 GND)
- 버튼 입력: GPIO = HIGH (3.3V로 직결)
```

### 4. 분압기 (Voltage Divider)

```
5V ─── [R1: 10kΩ] ─── A0 (ADC 입력) ─── [R2: 10kΩ] ─── GND

출력 전압: Vout = Vin × R2 / (R1 + R2)
예: 5V × 10kΩ / (10kΩ + 10kΩ) = 2.5V
주의: ESP32 ADC는 최대 3.3V이므로 5V 이상 전압은 반드시 분압 필요
```

### 5. I2C 풀업 저항

```
3.3V ─── [4.7kΩ] ─── SDA
3.3V ─── [4.7kΩ] ─── SCL

참고: 많은 I2C 모듈에 내장되어 있음. 중복 연결 시 병렬로 저항값 낮아짐.
```

## 색상 코드 읽는 법

### 4색 띠 저항 (허용오차 ±5%, ±10%)

```
   [띠1][띠2][띠3][띠4]
    첫째  둘째 승수  오차
```

| 색상 | 숫자 | 승수 | 오차 |
|------|------|------|------|
| 검정 (Black) | 0 | ×10⁰ (×1) | — |
| 갈색 (Brown) | 1 | ×10¹ (×10) | ±1% |
| 빨강 (Red) | 2 | ×10² (×100) | ±2% |
| 주황 (Orange) | 3 | ×10³ (×1k) | — |
| 노랑 (Yellow) | 4 | ×10⁴ (×10k) | — |
| 초록 (Green) | 5 | ×10⁵ (×100k) | ±0.5% |
| 파랑 (Blue) | 6 | ×10⁶ (×1M) | ±0.25% |
| 보라 (Violet) | 7 | ×10⁷ | ±0.1% |
| 회색 (Gray) | 8 | ×10⁸ | ±0.05% |
| 흰색 (White) | 9 | ×10⁹ | — |
| 금색 (Gold) | — | ×0.1 | ±5% |
| 은색 (Silver) | — | ×0.01 | ±10% |

**예시 1**: 빨강-빨강-갈색-금색
→ 2, 2, ×10 = 220Ω ±5%

**예시 2**: 주황-주황-갈색-금색
→ 3, 3, ×10 = 330Ω ±5%

**예시 3**: 갈색-검정-주황-금색
→ 1, 0, ×1000 = 10,000Ω = 10kΩ ±5%

### 5색 띠 저항 (허용오차 ±1%)

```
   [띠1][띠2][띠3][띠4][띠5]
    첫째  둘째  셋째  승수  오차
```

**예시**: 갈색-검정-검정-빨강-갈색
→ 1, 0, 0, ×100 = 10,000Ω = 10kΩ ±1%

### SMD 저항 마킹 코드

**3자리 코드**: 앞 두 자리 = 유효 숫자, 마지막 자리 = 0의 개수
- `103` → 10 × 10³ = 10,000Ω = 10kΩ
- `472` → 47 × 10² = 4,700Ω = 4.7kΩ
- `221` → 22 × 10¹ = 220Ω

**4자리 코드** (정밀 저항):
- `1002` → 100 × 10² = 10,000Ω = 10kΩ
- `4991` → 499 × 10¹ = 4,990Ω ≈ 5kΩ

**EIA-96 코드** (최신 정밀 SMD):
- `01C` → 코드표 참조 필요

## 주의사항

### 1. 전력 초과 주의
- 정격 전력 이상의 전류가 흐르면 **과열 및 소손**됩니다.
- 아두이노 핀에서 나오는 전류(최대 40mA)와 저항값을 계산해서 P = I² × R 확인
- 안전 마진: 정격 전력의 **50% 이하**에서 사용 권장
- 예: 1/4W 저항은 실제로 0.125W(125mW) 이하로 사용

### 2. 저항값 오차
- ±5% 저항은 실제 측정값이 설계값과 다를 수 있음
- 정밀 회로(ADC, 분압기)에는 ±1% 금속 피막 저항 사용

### 3. 온도 상승에 따른 저항값 변화
- 고온에서는 저항값이 변동 (온도 계수 참고)
- 발열이 심한 부품 근처에는 설치 금지

### 4. 노이즈 민감 회로
- 탄소 피막 저항은 미세한 노이즈를 발생시킴
- 오디오, 고정밀 ADC 회로에는 금속 피막 저항 사용

### 5. 방향성 없음
- 저항에는 극성(방향)이 없으므로 어느 방향으로 연결해도 무방

## 데이터시트 주요 파라미터 설명

| 파라미터 | 영문명 | 설명 |
|----------|--------|------|
| 저항값 | Resistance (R) | 기본 저항값, 단위 Ω |
| 허용 오차 | Tolerance | 실제값의 오차 범위 (±%) |
| 정격 전력 | Rated Power (P) | 안전하게 소모할 수 있는 최대 전력 |
| 최대 전압 | Maximum Voltage | 양단에 걸 수 있는 최대 전압 |
| 온도 계수 | Temperature Coefficient (TCR) | 온도 변화에 따른 저항값 변화율 (ppm/°C) |
| 동작 온도 범위 | Operating Temperature Range | 정상 동작 가능한 온도 범위 |
| 노이즈 지수 | Noise Index | 전류 노이즈 크기 (dB) |
| 전압 계수 | Voltage Coefficient | 전압에 따른 저항값 변화 (ppm/V) |
| 단락 내성 | Overload Voltage | 단기 과부하 허용 전압 |
| 절연 저항 | Insulation Resistance | 저항 외장과 리드 간 절연 (GΩ) |
