# AMS1117 LDO 전압 레귤레이터

## 개요

AMS1117은 Advanced Monolithic Systems에서 제조한 저손실(LDO, Low Dropout) 선형 전압 레귤레이터입니다.
입력 전압이 출력 전압보다 약간만 높아도 안정적으로 동작하는 것이 특징으로,
드롭아웃 전압이 최대 1.3V(1A 부하 기준)로 낮아 배터리 구동 기기나 USB 전원에 적합합니다.

Arduino 프로젝트에서는 주로 다음 용도로 사용합니다.
- 5V → 3.3V 변환 (ESP32, ESP8266, 센서 모듈 등 3.3V 계열 부품 구동)
- USB 5V → 3.3V 변환 (AMS1117-3.3 내장 브레이크아웃 모듈)
- 소전류 보조 전원 회로

## 핀 구성

### SOT-223 패키지 (가장 일반적)

```
   [ 1 ]  [ 2 ]  [ 3 ]
  GND   OUTPUT  INPUT
        (탭=OUTPUT)
```

| 핀 번호 | 핀 이름 | 기능 |
|--------|--------|------|
| 1 | GND | 접지. 회로 공통 기준 전위 |
| 2 (탭 포함) | OUTPUT | 조정된 출력 전압. SOT-223 탭은 핀2와 내부 연결 |
| 3 | INPUT | 원시 입력 전압 인가 |

### TO-252 (D-PAK) 패키지

| 핀 번호 | 핀 이름 | 기능 |
|--------|--------|------|
| 1 | ADJ/GND | ADJ 모델: 조정 핀 / 고정 모델: GND |
| 2 (탭) | OUTPUT | 출력 전압, 탭은 방열 |
| 3 | INPUT | 입력 전압 |

### AMS1117-ADJ 가변형 핀 배열

| 핀 | 기능 |
|----|------|
| 1 | ADJ (조정 핀, 외부 분압 저항 연결) |
| 2 | OUTPUT |
| 3 | INPUT |

## 동작 원리

AMS1117은 내부 기준 전압(Vref ≈ 1.25V)과 오차 증폭기(Error Amplifier), 패스 트랜지스터(PNP MOSFET)로 구성됩니다.

1. 출력 전압이 목표값보다 낮아지면 → 오차 증폭기가 패스 트랜지스터를 더 열어 전류 증가
2. 출력 전압이 목표값보다 높아지면 → 패스 트랜지스터를 조금 닫아 전류 감소
3. 이 피드백 루프가 출력 전압을 일정하게 유지

LDO의 핵심은 "패스 트랜지스터의 포화 전압이 낮다"는 점으로, 일반 선형 레귤레이터(LM7805 등)보다 Vin − Vout 최솟값이 현저히 낮습니다.

**발열 계산**:
```
P_발열 = (Vin − Vout) × Iout
예) 5V → 3.3V, 500mA: P = (5 − 3.3) × 0.5 = 0.85W
```

## 아두이노 연결 방법

### 기본 3.3V 레귤레이터 회로

```
        C1(10μF)        C2(10μF)
Vin ────┤+        +├──── Vout(3.3V)
        └── GND ──┘
        │                 │
        └── AMS1117-3.3 ──┘
            INPUT → OUTPUT
```

**실제 배선**:
```
전원(5V) ──── AMS1117 INPUT(핀3) ──── 10μF 캐패시터(+ 출력측, - GND)
               AMS1117 OUTPUT(핀2) ── 10μF 캐패시터(+ 출력측, - GND) ── 3.3V 부하
               AMS1117 GND(핀1) ────── 공통 GND
```

### AMS1117-ADJ 가변형 회로

```
            R1
OUTPUT ──── [240Ω] ──── ADJ ──── [R2] ──── GND
                         │
                        Cadj(10μF)
```

출력 전압 계산:
```
Vout = Vref × (1 + R2/R1) + Iadj × R2
     ≈ 1.25 × (1 + R2/240)   (Iadj ≈ 50μA 이므로 근사)

목표 전압별 R2 값:
  1.8V: R2 ≈ 259Ω (270Ω 표준값 사용)
  2.5V: R2 ≈ 480Ω (470Ω 표준값 사용)
  3.3V: R2 ≈ 383Ω (390Ω 표준값 사용)
  5.0V: R2 ≈ 720Ω (750Ω 표준값 사용)
```

### ESP32-C3 와 연결 예시

```
USB 5V ─── AMS1117-3.3 INPUT
           AMS1117-3.3 OUTPUT ─── ESP32-C3 3V3 핀
           AMS1117-3.3 GND ────── ESP32-C3 GND 핀
```

## 주의사항

1. **발열 관리**: 고전압 입력(예: 12V)에 고전류 부하를 사용하면 발열이 매우 심합니다. 방열판 부착 또는 전류를 300mA 이하로 제한하세요.
2. **캐패시터 필수**: 입력/출력 양단에 각각 10μF 전해 캐패시터 + 100nF 세라믹 캐패시터를 병렬로 달아야 안정적으로 동작합니다. 캐패시터 없으면 발진(oscillation)이 발생할 수 있습니다.
3. **최소 입력 전압 확보**: 드롭아웃 전압을 반드시 확보하세요. USB 5V → 3.3V 변환 시 케이블 손실 등으로 USB 전압이 4.5V로 떨어지면 출력이 불안정해집니다.
4. **역전압 보호 없음**: 입출력이 반대로 연결되면 소자가 즉시 파손됩니다.
5. **출력 단락**: 단락 보호 회로가 있지만 반복 단락은 소자를 열화시킵니다.

## 실사용 팁 및 회로 패턴

### 입출력 캐패시터 권장 구성

```
입력측: 10μF 전해 + 100nF 세라믹 (병렬)
출력측: 10μF 전해 + 100nF 세라믹 (병렬)
```
- 전해 캐패시터: 저주파 노이즈 제거 및 안정화
- 세라믹 캐패시터: 고주파 노이즈 제거 (바이패스)

### 방열판 적용 기준

| 부하 전류 | 입력 전압 | 방열판 필요 여부 |
|----------|----------|----------------|
| < 200mA | < 6V | 불필요 |
| 200~500mA | 5V | 선택적 |
| 200~500mA | 9~12V | 필수 |
| > 500mA | 모든 전압 | 필수 |

### 병렬 연결로 전류 증가

동일 로트 AMS1117 2개를 병렬 연결하면 이론상 전류가 2배 되지만, 개체 편차로 인해 부하 분담이 불균형해질 수 있습니다.
→ 각 IC 출력에 0.1Ω 전류 균형 저항을 직렬 추가하면 개선됩니다.

### Arduino 스케치 연동

```cpp
// config.h
#define PIN_3V3_ENABLE 5  // AMS1117 EN 핀 있는 모듈 사용 시
                           // EN LOW = 출력 차단, EN HIGH = 출력 활성

// 저전력 모드: 3.3V 부하 전원 차단
void disablePeripherals() {
    digitalWrite(PIN_3V3_ENABLE, LOW);
}
```
