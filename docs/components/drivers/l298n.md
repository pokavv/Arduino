# L298N 모터 드라이버

## 개요

L298N은 ST Microelectronics가 제조한 고전압·고전류 듀얼 H-브리지 드라이버 IC입니다. 하나의 칩으로 DC 모터 2개, 또는 스텝 모터 1개를 독립적으로 양방향 구동할 수 있습니다. 아두이노 프로젝트에서 가장 널리 사용되는 모터 드라이버 모듈의 핵심 IC입니다.

**H-브리지(H-Bridge)**: 4개의 스위치(트랜지스터)를 H자 형태로 배치하여 모터에 양방향 전류를 흘릴 수 있는 회로 구조.

---

## 핀/단자 구성

### L298N IC (멀티웟 패키지, 15핀)

| 핀 번호 | 핀 이름 | 기능 |
|--------|---------|------|
| 1 | SENSE A | 전류 감지 저항 A (보통 GND에 연결) |
| 2 | OUT1 | 모터 A 출력 1 |
| 3 | OUT2 | 모터 A 출력 2 |
| 4 | VS | 모터 전원 (+), 최대 46V |
| 5 | IN1 | 모터 A 방향 제어 입력 1 |
| 6 | ENA | 모터 A 활성화 (PWM 속도 제어) |
| 7 | IN2 | 모터 A 방향 제어 입력 2 |
| 8 | GND | 접지 |
| 9 | VSS | 로직 전원 (4.5 ~ 7V) |
| 10 | IN3 | 모터 B 방향 제어 입력 3 |
| 11 | ENB | 모터 B 활성화 (PWM 속도 제어) |
| 12 | IN4 | 모터 B 방향 제어 입력 4 |
| 13 | OUT3 | 모터 B 출력 3 |
| 14 | OUT4 | 모터 B 출력 4 |
| 15 | SENSE B | 전류 감지 저항 B |

### L298N 모듈 (브레이크아웃 보드) 단자

| 단자 | 기능 |
|------|------|
| VS (12V 또는 6~12V) | 모터 전원 입력 |
| GND | 공통 접지 |
| 5V | 5V 출력 (온보드 레귤레이터, 모터 전원 사용 시) |
| ENA | 모터 A 활성화/PWM (점퍼 제거 후 연결) |
| IN1 | 모터 A 방향 제어 1 |
| IN2 | 모터 A 방향 제어 2 |
| OUT1, OUT2 | 모터 A 출력 |
| IN3 | 모터 B 방향 제어 3 |
| IN4 | 모터 B 방향 제어 4 |
| ENB | 모터 B 활성화/PWM |
| OUT3, OUT4 | 모터 B 출력 |

## 전기적 스펙 표

| 항목 | 값 |
|------|----|
| 모터 전원 전압 (VS) | 4.5 ~ 46 V |
| 로직 전원 전압 (VSS) | 4.5 ~ 7 V |
| 로직 입력 전압 (HIGH) | 2.3 ~ VSS V |
| 로직 입력 전압 (LOW) | -0.3 ~ 1.5 V |
| 출력 전류 (채널당) | 2 A (연속), 3 A (피크) |
| 총 소비 전력 | 25 W (패키지 한계) |
| 내부 포화 전압 드롭 | ~2 V (각 H-브리지 측) |
| 전환 주파수 | 최대 40 kHz (PWM) |
| 열 저항 (Rth j-amb) | 35 °C/W (멀티와트 패키지) |
| 동작 온도 | -25 ~ +130 °C |
| 방열판 필요 온도 | 70°C 이상 시 |

> **전압 드롭 주의**: L298N은 내부 트랜지스터 포화 전압으로 ~2V 손실. 5V 공급 시 모터에 실제 도달 전압 ≈ 3V 수준. 모터 본래 전압 + 2~3V를 공급해야 정격 성능.

## 동작 원리

H-브리지는 4개의 스위치(Q1~Q4)로 구성됩니다.

```
VS
 |
Q1(상좌)    Q3(상우)
 |              |
OUT1 ──[모터]── OUT2
 |              |
Q2(하좌)    Q4(하우)
 |
GND
```

- **정방향**: Q1, Q4 ON → 전류: VS → Q1 → OUT1 → 모터 → OUT2 → Q4 → GND
- **역방향**: Q2, Q3 ON → 전류: VS → Q3 → OUT2 → 모터 → OUT1 → Q2 → GND
- **정지(Coast)**: Q1,Q2,Q3,Q4 모두 OFF → 모터 자유 회전
- **제동(Brake)**: Q1,Q3 ON 또는 Q2,Q4 ON → 모터 단락 → 급제동

## 진리표 (Truth Table)

### 모터 A (ENA = HIGH 고정)

| ENA | IN1 | IN2 | 동작 |
|-----|-----|-----|------|
| HIGH | HIGH | LOW | 정방향 회전 |
| HIGH | LOW | HIGH | 역방향 회전 |
| HIGH | LOW | LOW | 자유 회전 (Coast) |
| HIGH | HIGH | HIGH | 제동 (Brake) |
| LOW | X | X | 정지 (비활성) |
| PWM | HIGH | LOW | 정방향 속도 제어 |
| PWM | LOW | HIGH | 역방향 속도 제어 |

### 모터 B (ENB = HIGH 고정)

| ENB | IN3 | IN4 | 동작 |
|-----|-----|-----|------|
| HIGH | HIGH | LOW | 정방향 회전 |
| HIGH | LOW | HIGH | 역방향 회전 |
| HIGH | LOW | LOW | 자유 회전 |
| HIGH | HIGH | HIGH | 제동 |
| LOW | X | X | 정지 (비활성) |

## 아두이노 연결 방법

```
외부 전원 (6~12V, 2A+)
├── (+) ────────────────────→ L298N 모듈 VS (12V)
└── (─) ────────────────────→ L298N 모듈 GND

ESP32-C3 Super Mini
├── G4 ──────────────────→ IN1
├── G5 ──────────────────→ IN2
├── G6 (PWM) ────────────→ ENA  (점퍼 제거 후)
├── G7 ──────────────────→ IN3
├── G8 ──────────────────→ IN4
├── G9 (PWM) ────────────→ ENB  (점퍼 제거 후)
└── GND ─────────────────→ L298N GND

L298N 출력
├── OUT1, OUT2 ──────────→ 모터 A (+), (-)
└── OUT3, OUT4 ──────────→ 모터 B (+), (-)

L298N 5V 출력 (모터 전압 ≥ 7V 시 사용 가능)
└── 5V ──────────────────→ ESP32-C3 5V 핀 (선택)
```

> 모터 전원이 6V 미만이면 온보드 5V 레귤레이터 비활성. 로직 전원 5V는 별도 공급.  
> ESP32-C3는 3.3V 로직 → IN 핀 3.3V 신호 인식 가능 (LOW: 0~1.5V, HIGH: 2.3V~).

## 제어 방법 (아두이노 코드)

```cpp
// ────── 핀 정의 ──────
const int ENA_PIN = 6;
const int IN1_PIN = 4;
const int IN2_PIN = 5;
const int ENB_PIN = 9;
const int IN3_PIN = 7;
const int IN4_PIN = 8;

// ────── ledc 채널 ──────
const int CH_A = 0;  // ENA
const int CH_B = 1;  // ENB
const int PWM_FREQ = 1000;  // 1 kHz
const int PWM_RES  = 8;     // 8비트 (0~255)

void setup() {
  pinMode(IN1_PIN, OUTPUT);
  pinMode(IN2_PIN, OUTPUT);
  pinMode(IN3_PIN, OUTPUT);
  pinMode(IN4_PIN, OUTPUT);

  ledcSetup(CH_A, PWM_FREQ, PWM_RES);
  ledcSetup(CH_B, PWM_FREQ, PWM_RES);
  ledcAttachPin(ENA_PIN, CH_A);
  ledcAttachPin(ENB_PIN, CH_B);
}

// 모터 A 제어
void motorA(int speed) {
  // speed: -255 ~ +255 (음수 = 역방향)
  if (speed > 0) {
    digitalWrite(IN1_PIN, HIGH);
    digitalWrite(IN2_PIN, LOW);
    ledcWrite(CH_A, speed);
  } else if (speed < 0) {
    digitalWrite(IN1_PIN, LOW);
    digitalWrite(IN2_PIN, HIGH);
    ledcWrite(CH_A, -speed);
  } else {
    digitalWrite(IN1_PIN, LOW);
    digitalWrite(IN2_PIN, LOW);
    ledcWrite(CH_A, 0);
  }
}

// 모터 B 제어
void motorB(int speed) {
  if (speed > 0) {
    digitalWrite(IN3_PIN, HIGH);
    digitalWrite(IN4_PIN, LOW);
    ledcWrite(CH_B, speed);
  } else if (speed < 0) {
    digitalWrite(IN3_PIN, LOW);
    digitalWrite(IN4_PIN, HIGH);
    ledcWrite(CH_B, -speed);
  } else {
    digitalWrite(IN3_PIN, LOW);
    digitalWrite(IN4_PIN, LOW);
    ledcWrite(CH_B, 0);
  }
}

// 능동 브레이크
void brakeA() {
  digitalWrite(IN1_PIN, HIGH);
  digitalWrite(IN2_PIN, HIGH);
  ledcWrite(CH_A, 255);
}

void loop() {
  motorA(200);   // A 정방향 80%
  motorB(-150);  // B 역방향 59%
  delay(2000);
  motorA(0);
  motorB(0);
  delay(500);
}
```

## 주의사항

1. **발열**: L298N은 효율이 낮아 (~60~70%) 고전류 시 심한 발열. 방열판 필수. 장시간 2A 이상 구동 금지.
2. **전압 드롭**: 내부 ~2V 드롭. 모터 정격 5V → 최소 7V 입력 필요.
3. **점퍼 핀**: 모듈의 ENA/ENB 점퍼를 제거해야 PWM 속도 제어 가능. 점퍼 연결 시 항상 최대 속도.
4. **PWM 주파수**: 너무 높으면 L298N 내부 스위칭 손실 증가 → 1~20 kHz 범위 권장.
5. **플라이백 다이오드**: L298N IC에 내장. 별도 추가 불필요.
6. **최소 동작 전압**: 모터 전압이 너무 낮으면 모터 미동작. 기동 시 최소 PWM 값 조정 필요.
7. **스텝 모터 연결**: 4선 스텝 모터(바이폴라) OUT1~OUT4에 연결 가능. ENA/ENB는 HIGH 고정.

```cpp
// L298N으로 바이폴라 스텝 모터 구동 (4선)

```

### 패턴 2: 2WD 로봇 전진/후진/회전

```cpp
void forward(int spd)  { motorA(spd);   motorB(spd);   }
void backward(int spd) { motorA(-spd);  motorB(-spd);  }
void turnLeft(int spd) { motorA(spd/2); motorB(spd);   }
void turnRight(int spd){ motorA(spd);   motorB(spd/2); }
void stop()            { motorA(0);     motorB(0);     }
```

### 패턴 3: 가속/감속 소프트 스타트

```cpp
void rampUp(int targetSpeed, int stepDelay) {
  for (int s = 0; s <= targetSpeed; s += 10) {
    motorA(s);
    delay(stepDelay);
  }
}
```
