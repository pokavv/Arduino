# 핀 헤더 / 소켓 헤더

## 개요

핀 헤더(Pin Header)는 PCB에 납땜하여 전선, 모듈, 커넥터를 연결하는 데 사용하는 금속 핀 어레이 부품입니다.
소켓 헤더(Socket Header / Female Header)는 핀 헤더의 암 대응 부품으로, 핀 헤더를 수용하여 탈착 가능한 연결을 만듭니다.

Arduino 프로젝트에서는 다음 목적으로 사용합니다.
- Arduino 보드의 확장 핀 헤더 (Uno R3의 쉴드 연결용 소켓)
- 센서/모듈 보드의 인터페이스 핀
- 커스텀 PCB와 모듈 간 연결
- 점퍼 와이어 연결 포인트
- 브레이크아웃 보드 제작

## 핀 구성 (표준 2.54mm 단열 핀 헤더)

```
핀 헤더 단면:
┌──────────┐
│  플라스틱 │← 기저 높이 약 2.5mm
│  하우징  │
└──┬────┬──┘
   │핀 1│← 납땜 다리 (약 3mm)
   │    │← 삽입 부분 (약 8~11mm)
   └────┘
```

| 부분 | 치수 | 재질 |
|------|------|------|
| 핀 직경 | 0.64mm | 황동 + 금 또는 주석 도금 |
| 핀 피치 | 2.54mm | — |
| 하우징 높이 | 2.5mm | 나일론(UL94V-0) |
| 납땜 다리 길이 | 3.0mm | — |
| 삽입부 길이(표준) | 8.5mm | — |
| 삽입부 길이(롱) | 11.0mm | — |

## 동작 원리

핀 헤더와 소켓의 접촉 방식:

```
소켓 내부:
┌─────────────────┐
│ [ 스프링 접촉자 ] │
│   ↑               │
│ 핀 삽입 시 클립이 │
│ 벌어지며 마찰 접촉 │
└─────────────────┘
```

전기 전도:
핀 표면 도금(금/주석) → 소켓 스프링 접촉자 → 도금 → 기판 패드 → 납땜

금 도금이 주석 도금보다 접촉 저항이 낮고 산화에 강하지만 가격이 높습니다.
Arduino 프로젝트 수준에서는 주석 도금으로 충분합니다.

## 아두이노 연결 방법

### 핀 헤더 납땜 방법 (자작 PCB/모듈)

```
준비물:
  핀 헤더, PCB, 납땜 인두(350~370°C), 납(Sn63/Pb37 또는 무연)

순서:
1. PCB 홀에 핀 헤더 삽입
2. 임시 고정 (블루택 또는 브레드보드에 꽂아 수직 유지)
3. 인두를 핀과 패드가 만나는 지점에 2~3초 접촉
4. 납을 핀과 패드 사이에 공급
5. 인두 제거 후 3~5초 고정 유지 (납이 굳는 동안 흔들지 말 것)
6. 광택 있는 납 표면 확인 (무광택 = 냉납, 재납땜 필요)
```

### Arduino Nano에 핀 헤더 설치

```
핀 헤더 (15핀 × 2줄) 또는 (14핀 양쪽)
→ 브레드보드에 꽂아 위치 고정
→ Nano 보드를 위에서 끼워 정렬
→ 한쪽 끝 1~2핀 납땜 후 수직 확인
→ 나머지 핀 납땜
```

### ESP32-C3 Super Mini 핀 헤더

```
ESP32-C3 Super Mini: 핀 헤더 미설치 상태로 판매
→ 2.54mm 단열 핀 헤더 11핀 × 2줄 준비
→ 브레드보드에 핀 헤더 꽂고 보드 올려 납땜
→ 납땜 후 브레드보드에서 제거
```

### 탈착 가능한 모듈 연결 (소켓 헤더 사용)

```
주 보드에 소켓 헤더 납땜
    ↓
센서/모듈 보드에 핀 헤더 납땜
    ↓
핀 헤더를 소켓에 삽입 → 탈착 가능 연결
```

## 주의사항

1. **피치 혼용 금지**: 2.54mm 핀 헤더를 2.00mm 소켓에 억지로 삽입하면 핀이 구부러지고 소켓이 파손됩니다. 반드시 같은 피치끼리 연결.
2. **냉납(Cold Joint) 방지**: 인두 온도가 낮거나 납땜 시간이 짧으면 냉납이 발생하여 접촉 불량. 납 표면이 광택 없이 회백색이면 냉납.
3. **납땜 플럭스 잔여물**: 산성 플럭스(수용성)는 납땜 후 IPA로 세정해야 합니다. 비세정(No-Clean) 플럭스 납은 세정 생략 가능.
4. **열에 의한 소켓 변형**: 납땜 시 플라스틱 하우징이 열로 변형될 수 있습니다. 핀당 납땜 시간을 3초 이내로 제한하고, 인접 핀은 식힌 후 납땜.
5. **삽입 방향**: 2열 핀 헤더는 키(홈, Key) 없이 대칭인 경우가 많습니다. 역방향 삽입을 방지하기 위해 실크스크린 표시나 핀 1 위치를 명확히 표기.
6. **반복 탈착 한계**: 표준 핀 헤더는 삽발 수명이 약 30회 수준. 자주 교체하는 모듈은 고급형(금 도금, 수명 300회) 사용 권장.

## 실사용 팁 및 회로 패턴

### 핀 헤더 자르기

```
단열 핀 헤더는 원하는 핀 수로 쉽게 자를 수 있습니다.
방법:
  니퍼(Wire Cutter)로 하우징 플라스틱을 원하는 위치에서 자름
  또는
  2열 헤더는 롱노즈 플라이어로 끊고 싶은 핀을 구부려 떼어냄

팁: 자를 때 1핀을 여유분으로 남기고 자른 후, 남은 1핀을 니퍼로 제거
    → 하우징 갈라짐 방지
```

### 쉴드 스택을 위한 롱 소켓

```
Arduino Uno + 쉴드 스택:
  Uno ← 롱 소켓 헤더(11mm) 납땜
      ← 쉴드 A 핀 헤더 삽입
      ← 쉴드 A에도 롱 소켓 납땜
      ← 쉴드 B 핀 헤더 삽입
  → 최대 3~4겹 스택 가능
```

### 납땜 없이 테스트 (IC 테스트 소켓)

```
DIP IC용 테스트 소켓:
  ZIF(Zero Insertion Force) 소켓 사용 시
  레버로 고정/해제 → 납땜 없이 IC 테스트
```
