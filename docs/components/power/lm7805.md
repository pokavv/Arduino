# LM7805 선형 전압 레귤레이터 5V

## 개요

LM7805는 Texas Instruments(구 National Semiconductor)에서 개발한 고정 출력 +5V 선형 전압 레귤레이터입니다.
78xx 시리즈 중 가장 널리 사용되는 모델로, 수십 년간 전자 회로의 표준 5V 전원 공급 부품으로 자리잡았습니다.

드롭아웃 전압이 약 2V로 높아(최소 입력 7V 필요) LDO는 아니지만,
가격이 매우 저렴하고 전 세계 어디서나 구할 수 있으며 사용법이 단순합니다.

Arduino 프로젝트에서는 주로 다음 용도로 사용합니다.
- 9V 배터리 → 5V 변환 (Arduino Uno, Nano 등 5V 계열 보드 구동)
- 12V 어댑터 → 5V 변환 (외부 전원 공급)
- 릴레이, 모터 드라이버, 5V 센서 모듈 전원 공급
- 교육·프로토타입 회로의 간단한 전원 공급

## 핀 구성

### TO-220 패키지 (표준, 가장 일반적)

```
앞면(금속 탭이 뒤)에서 봤을 때:
  [ 1 ]  [ 2 ]  [ 3 ]
  INPUT  GND  OUTPUT
```

| 핀 번호 | 핀 이름 | 기능 |
|--------|--------|------|
| 1 | INPUT | 원시 입력 전압 (7V ~ 35V) |
| 2 | GND | 공통 접지 |
| 3 | OUTPUT | 조정된 5V 출력 |
| 탭(금속) | GND | 핀2(GND)와 내부 연결, 방열판 부착 위치 |

### TO-92 패키지 (LM78L05, 소형)

```
평평한 면을 앞으로 했을 때:
  [ 1 ]  [ 2 ]  [ 3 ]
 OUTPUT  GND  INPUT
```
TO-220과 핀 배열이 다르니 주의!

### TO-3 패키지 (고전류용)

| 핀 | 기능 |
|----|------|
| 1 | INPUT |
| 2 | OUTPUT |
| 케이스(금속 하우징) | GND |

## 동작 원리

LM7805 내부는 크게 세 블록으로 구성됩니다.

1. **기준 전압 회로(Bandgap Reference)**: 온도 변화에 안정적인 내부 기준 전압(약 5V) 생성
2. **오차 증폭기(Error Amplifier)**: 출력 전압을 기준 전압과 비교하여 차이를 검출
3. **직렬 패스 트랜지스터(Series Pass Transistor)**: 오차 증폭기 신호로 제어되어 출력 전류 조절

**발열 계산**:
```
P_발열 = (Vin − Vout) × Iout
예) 9V → 5V, 500mA: P = (9 − 5) × 0.5 = 2.0W
예) 12V → 5V, 1A:   P = (12 − 5) × 1.0 = 7.0W  ← 방열판 필수
```

과열 시 내부 열차단 회로(Thermal Shutdown)가 동작하여 출력을 차단하고 소자를 보호합니다.

## 아두이노 연결 방법

### 기본 5V 전원 회로

```
전원(9V~12V) ─── LM7805 INPUT(핀1)
                  LM7805 GND(핀2) ──── 공통 GND
                  LM7805 OUTPUT(핀3) ── 5V 부하 (Arduino VIN 또는 5V 핀)
```

**캐패시터 포함 완전한 회로**:
```
Vin ──┬─── [LM7805 INPUT]──[LM7805 OUTPUT]───┬─── Vout(5V)
      │                                        │
    [C1]                                     [C2]
    0.33μF                                  0.1μF
      │                                        │
GND ──┴─────────── [LM7805 GND] ──────────────┴─── GND
```

| 캐패시터 | 값 | 역할 |
|--------|-----|------|
| C1 (입력측) | 0.33μF (또는 10μF) | 입력 안정화, LM7805 거리가 멀 때 필수 |
| C2 (출력측) | 0.1μF (또는 10μF) | 출력 과도 응답 개선 |

### Arduino Uno 외부 전원 공급

```
12V 어댑터 ─── LM7805 → Arduino Uno VIN 핀
              또는 Arduino Uno 배럴 잭 직결 (내장 레귤레이터 사용)
```

### 고전류 회로 (방열판 부착)

```
       방열판(알루미늄)
          │
TO-220 탭 부착 (절연 패드 사용 시 절연)
          │
LM7805 → 출력 1A까지 안전하게 사용
```

## 주의사항

1. **발열 심각 주의**: 12V 입력, 1A 출력 시 7W 발열. 방열판 없이는 10초 이내에 열차단 동작. 반드시 방열판을 부착하고 서멀 컴파운드(실리콘 그리스)를 도포하세요.
2. **최소 입력 전압 7V**: USB 5V를 입력으로 사용하면 드롭아웃으로 인해 출력이 5V 미달됩니다. 입력은 반드시 7V 이상.
3. **역전압 없음**: 역방향 연결 시 IC 파손. 입력에 1N4001 다이오드 직렬 연결로 보호 가능(단, 0.7V 전압 강하 발생).
4. **출력 캐패시터 극성**: 전해 캐패시터(유극성)는 + 극을 출력 쪽에 연결.
5. **TO-220 탭은 GND**: 방열판이 섀시(금속 케이스)에 닿을 경우 GND가 케이스에 연결됩니다. 절연이 필요하면 절연 패드와 절연 와셔 사용.
6. **3.3V 부품에 직접 연결 금지**: 5V 출력을 ESP32, ESP8266 등 3.3V 부품에 직결하면 소손.

## 실사용 팁 및 회로 패턴

### 9V 배터리로 Arduino 구동

```
9V 배터리 → LM7805 → Arduino Uno VIN(5V)

예상 지속 시간 계산:
  9V 배터리 용량: 약 500mAh
  Arduino 소비 전류: 약 50mA
  발열 소비: (9−5) × 0.05 = 0.2W → 무시할 수준
  → 약 500/50 = 10시간 이론상 가능
  → 실제는 배터리 특성상 7~8시간
```

### 5V + 3.3V 듀얼 전원 공급 (AMS1117 조합)

```
12V 전원 ─── LM7805 ─── 5V ─── Arduino 5V 부품
                         │
                      AMS1117-3.3 ─── 3.3V ─── ESP32, 센서 모듈
```

### 출력 전압 약간 높이기 (비표준, 참고용)

```
GND 핀에 다이오드를 직렬 삽입:
LM7805 GND ─── [1N4001] ─── 실제 GND
→ 다이오드 전압 강하(0.7V)만큼 출력 전압 상승: 5.0V + 0.7V = 5.7V
→ 2개 직렬: 5.0V + 1.4V = 6.4V (비표준, 오차 있음)
```

### 열차단 후 자동 복구

LM7805의 열차단은 자동 복구형이므로, 방열이 충분하면 냉각 후 자동으로 출력이 재개됩니다.
그러나 반복적인 열차단은 소자 수명을 단축시키므로, 근본적인 방열 설계를 개선해야 합니다.

### Arduino 스케치 (전원 모니터링)

```cpp
// A0에 분압 회로로 입력 전압 모니터링
// LM7805 입력 전압 감시 예제
const int PIN_VOLT_SENSE = A0;
const float R1 = 10000.0;  // 분압 저항 R1 (10kΩ)
const float R2 = 4700.0;   // 분압 저항 R2 (4.7kΩ)
const float VREF = 5.0;    // Arduino 기준 전압 (5V 계열)

void setup() {
    Serial.begin(115200);
}

void loop() {
    int raw = analogRead(PIN_VOLT_SENSE);
    float vSense = raw * (VREF / 1023.0);
    float vIn = vSense * (R1 + R2) / R2;
    Serial.print("입력 전압: ");
    Serial.print(vIn, 2);
    Serial.println("V");
    delay(1000);
}
```
