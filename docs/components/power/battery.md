# 배터리 종류 비교

## 개요

배터리는 Arduino 프로젝트를 콘센트 없이 독립적으로 동작시키는 핵심 전원 부품입니다.
배터리 종류마다 전압, 용량, 방전 특성, 충전 가능 여부가 다르므로 프로젝트 요구사항에 맞는 선택이 중요합니다.

Arduino 프로젝트에서 배터리가 필요한 상황:
- 이동형 로봇, 드론, RC카
- 야외 IoT 센서 (기상, 농업, 환경 모니터링)
- 웨어러블 기기
- 정전 대비 백업 전원
- 배터리로만 동작하는 독립형 장치

## 리튬이온 18650

### 개요

지름 18mm, 길이 65mm의 원통형 리튬이온 배터리. Arduino 장기 구동 프로젝트에서 가장 많이 사용됩니다.

### 스펙 표

| 항목 | 값 | 단위 |
|------|-----|------|
| 공칭 전압 | 3.6 ~ 3.7 | V |
| 만충 전압 | 4.2 | V |
| 방전 컷오프 전압 | 2.5 ~ 3.0 | V |
| 용량 범위 | 1800 ~ 3500 | mAh |
| 대표 용량 (Panasonic NCR18650B) | 3400 | mAh |
| 표준 충전 전류 | 0.5C (= 1700mA for 3400mAh) | mA |
| 최대 충전 전류 | 1C | mA |
| 최대 방전 전류 (표준) | 2C | A |
| 내부 저항 | 50 ~ 250 | mΩ |
| 사이클 수명 | 300 ~ 1000 | 회 |
| 동작 온도 (방전) | −20 ~ 60 | °C |
| 동작 온도 (충전) | 0 ~ 45 | °C |
| 크기 | φ18.6 × 65.2 | mm |
| 무게 | 44 ~ 48 | g |

### 보호 회로 (PCB 내장 여부)

| 종류 | 설명 | 사용 가능 여부 |
|------|------|--------------|
| 비보호형 (Unprotected) | PCB 없음, 순수 셀 | 보호 회로 외장 필수 |
| 보호형 (Protected) | 과충전/과방전/단락 보호 PCB 내장 | 안전, 일부 슬롯에 길이 문제 |

### Arduino 연결 방법

```
[18650 셀] → 보호 PCB 모듈(TP4056 등) → 5V 부스트 모듈 → Arduino 5V
          또는
[18650 셀] → 3.7V 직결 → ESP32 BAT 핀 (3.3~4.2V 입력 허용 시)
          또는
[18650 × 2 직렬] → 7.4V → LM7805 → 5V → Arduino VIN
```

## LiPo (리튬 폴리머)

### 개요

알루미늄 파우치에 젤 형태 전해질을 넣은 배터리. 드론, RC카, 웨어러블에 많이 사용됩니다.
18650 대비 얇고 다양한 모양으로 제작 가능하지만, 과충전/물리적 손상 시 발화 위험이 있습니다.

### 스펙 표

| 항목 | 값 | 단위 |
|------|-----|------|
| 셀 공칭 전압 | 3.7 | V |
| 셀 만충 전압 | 4.2 | V |
| 셀 방전 컷오프 | 3.0 | V |
| 용량 범위 | 100 ~ 10000+ | mAh |
| C 레이트 (방전) | 1C ~ 100C+ | — |
| 충전 C 레이트 | 0.5 ~ 5C | — |
| 사이클 수명 | 200 ~ 600 | 회 |
| 동작 온도 | −20 ~ 60 | °C |

### 셀 수(S) 및 전압

| 구성 | 공칭 전압 | 만충 전압 | 방전 컷오프 |
|------|----------|----------|------------|
| 1S | 3.7V | 4.2V | 3.0V |
| 2S | 7.4V | 8.4V | 6.0V |
| 3S | 11.1V | 12.6V | 9.0V |
| 4S | 14.8V | 16.8V | 12.0V |

### 주의사항

- 충전 전용 전용 밸런스 충전기 사용 필수 (B3, iMAX B6 등)
- 보관 시 50% 충전 상태 유지 (스토리지 모드)
- 부풀어 오른 LiPo는 즉시 사용 중지 및 안전 폐기
- 정격 C 레이트 초과 방전 금지

## AA 알카라인 배터리

### 개요

전 세계에서 가장 유통량이 많은 일회용 배터리. 낮은 가격과 편리한 구매가 장점이지만,
방전이 진행될수록 전압이 점진적으로 낮아지는 특성이 있습니다.

### 스펙 표

| 항목 | 값 | 단위 |
|------|-----|------|
| 공칭 전압 | 1.5 | V |
| 만충 전압 (새 배터리) | 1.6 | V |
| 방전 컷오프 | 0.8 ~ 1.0 | V |
| 용량 (표준 방전) | 2400 ~ 3000 | mAh |
| 내부 저항 (새 것) | 150 ~ 300 | mΩ |
| 자가 방전율 | 0.3 ~ 0.5 | %/월 |
| 저온 특성 | −18°C 이하 성능 급감 | — |
| 보관 수명 | 5 ~ 10 | 년 |

### Arduino 조합

```
AA × 4 = 6.0V → Arduino VIN (7V 미만이지만 일부 보드 동작)
AA × 4 + 레귤레이터 = 6V → LM7805(NG, 드롭아웃 부족) → AMS1117-5.0(OK, 1V 마진)
AA × 6 = 9.0V → Arduino VIN (권장)
AA × 4 = 6.0V → ESP32-C3 직결은 과전압 위험! 레귤레이터 필수
```

## 9V 알카라인 배터리

### 개요

6개의 AAAA 셀을 직렬 연결한 직사각형 배터리 (6F22). 실제 용량은 AA 배터리보다 훨씬 적습니다.
Arduino UNO의 DC 잭에 직결 가능하여 간편하지만, 용량이 적어 장시간 사용에 부적합합니다.

### 스펙 표

| 항목 | 값 | 단위 |
|------|-----|------|
| 공칭 전압 | 9.0 | V |
| 용량 (저전류) | 400 ~ 600 | mAh |
| 최대 연속 전류 | 50 ~ 100 | mA |
| 내부 저항 (새 것) | 1 ~ 3 | Ω |
| 내부 저항 (방전 중) | 5 ~ 20 | Ω |

### 주의사항

```
Arduino UNO(내장 LM1117-5.0 사용): 약 50mA 소비
→ 9V 500mAh → 500/50 = 10시간 이론상 가능
→ 실제: 내부 저항 증가로 5~7시간

모터/서보 추가 시: 순간 전류 1A+ → 내부 저항 전압 강하 → Arduino 리셋
→ 9V 배터리는 Arduino 단독 동작 외에는 부적합
```

## 충전 모듈 조합

### 18650 / LiPo 1S 충전 (TP4056 모듈)

```
TP4056 모듈 (USB Micro/Type-C 입력)
  - PROG 저항으로 충전 전류 설정 (기본 1A)
  - 과충전(4.2V), 과방전(2.5V) 보호
  - 충전 중: 빨간 LED
  - 완충: 파란 LED

연결:
  USB → TP4056 → 18650 셀
  18650 OUT+ → 5V 부스트(MT3608) → 5V → Arduino
```

### 배터리 전압 모니터링 (Arduino 코드)

```cpp
// 18650 셀 전압 모니터링
// 배터리(+) → [100kΩ] → [47kΩ] → GND
// 분압점 → Arduino A0 (3.3V ADC 기준 시)

const float R1 = 100000.0;
const float R2 = 47000.0;
const float VREF = 3.3;

float readBatteryVoltage(int pin) {
    int raw = analogRead(pin);
    float vSense = raw * (VREF / 4095.0);  // ESP32 12비트 ADC
    return vSense * (R1 + R2) / R2;
}

void checkBattery() {
    float v = readBatteryVoltage(A0);
    if (v < 3.3) {
        Serial.println("경고: 배터리 부족! 즉시 충전하세요.");
    } else if (v < 3.6) {
        Serial.println("배터리 잔량 낮음.");
    }
}
```
