# 변수, 자료형, 상수

## 변수가 뭔가요?

변수는 데이터를 저장하는 이름 붙은 공간입니다.
C/C++은 변수를 만들 때 **어떤 종류의 데이터인지(자료형)를 반드시 명시**해야 합니다.

```
자료형  변수이름 = 초기값;
  ↓        ↓        ↓
 int    온도값  =   25;
```

---

## 자료형 종류

### 정수형

| 자료형 | 크기 | 범위 | 사용 예 |
|--------|------|------|---------|
| `bool` | 1바이트 | true / false | 스위치 상태, 켜짐/꺼짐 |
| `byte` | 1바이트 | 0 ~ 255 | 밝기값, 작은 카운터 |
| `int` | 2바이트 | -32,768 ~ 32,767 | 일반 정수 |
| `unsigned int` | 2바이트 | 0 ~ 65,535 | 음수 없는 정수 |
| `long` | 4바이트 | 약 -21억 ~ 21억 | 큰 수, millis() 값 |
| `unsigned long` | 4바이트 | 0 ~ 약 42억 | 시간 계산 (millis 필수) |

### 실수형

| 자료형 | 크기 | 정밀도 | 사용 예 |
|--------|------|--------|---------|
| `float` | 4바이트 | 소수점 6~7자리 | 온도, 전압, 센서값 |
| `double` | 8바이트 | 소수점 15자리 | 고정밀 계산 (아두이노 우노에서는 float와 동일) |

### 문자형

| 자료형 | 크기 | 사용 예 |
|--------|------|---------|
| `char` | 1바이트 | 문자 하나 ('A', '1') |
| `String` | 가변 | 문자열 (아두이노 전용 클래스) |
| `char[]` | 가변 | C 스타일 문자열 배열 |

## 변수 선언과 초기화

```cpp
// 선언만 (값 미지정 — 쓰레기값이 들어있음, 위험!)
int 온도;

// 선언과 동시에 초기화 (권장)
int 온도 = 25;
float 전압 = 3.14;
bool 켜짐 = false;
char 등급 = 'A';
String 이름 = "ESP32";
```

---

## 자료형 선택 기준

```cpp
int 버튼상태 = 0;          // -3만 ~ 3만 범위의 정수
byte 밝기 = 255;           // 0 ~ 255 범위 (PWM값)
unsigned long 시간 = 0;    // millis() 는 반드시 unsigned long
float 온도 = 25.3;         // 소수점 필요
bool LED켜짐 = false;      // 참/거짓만
String 메시지 = "hello";   // 문자열
```

> **아두이노에서 `millis()`를 저장할 땐 반드시 `unsigned long` 사용!**
> `int`로 받으면 약 32초 후 오버플로우가 발생합니다.

## 상수 (const)

변수 앞에 `const`를 붙이면 값을 바꿀 수 없는 상수가 됩니다.
핀 번호, 고정된 설정값 등에 사용합니다.

```cpp
const int LED_PIN = 8;       // 핀 번호는 바뀌지 않으므로 const
const float PI = 3.14159;
const int 최대밝기 = 255;

// LED_PIN = 13;  ← 오류! 상수는 변경 불가
```

### const vs #define

```cpp
const int LED_PIN = 8;   // 타입이 있고, 디버거에서 확인 가능 — 권장
#define LED_PIN 8        // 타입 없이 단순 치환 — 구식 방식
```

## 변수 범위 (스코프)

변수가 어디에 선언됐냐에 따라 사용 가능한 범위가 달라집니다.

```cpp
int 전역변수 = 0;   // 파일 어디서든 사용 가능

void setup() {
    int 지역변수 = 10;  // setup() 안에서만 사용 가능
    전역변수 = 5;       // 전역변수는 접근 가능
}

void loop() {
    전역변수++;         // 전역변수는 접근 가능
    // 지역변수++;      // 오류! loop에서는 지역변수 접근 불가
}
```

### 전역변수를 써야 하는 경우
- `setup()`과 `loop()` 양쪽에서 모두 접근해야 할 때
- 인터럽트 함수와 공유해야 할 때 (`volatile` 필수)
- 함수 호출 사이에 값을 유지해야 할 때

## 형 변환 (캐스팅)

자료형을 다른 자료형으로 바꿉니다.

```cpp
int 정수값 = 10;
float 실수값 = (float)정수값;       // int → float: 10.0
int 내림 = (int)3.9;               // float → int: 3 (소수점 버림, 반올림 아님!)

int a = 7, b = 2;
float 결과 = (float)a / b;         // 3.5  (캐스팅 없으면 3이 됨!)
float 잘못된결과 = a / b;           // 3.0  (나눗셈 후 변환이라 소수점 손실)
```

## static 변수

함수 안에 선언하지만, **함수가 끝나도 값이 유지**됩니다.
카운터, 이전값 기억 등에 유용합니다.

```cpp
void loop() {
    static int 호출횟수 = 0;  // 처음 한 번만 0으로 초기화됨
    호출횟수++;
    Serial.println(호출횟수);  // 1, 2, 3, 4... 계속 증가
    delay(1000);
}
```
