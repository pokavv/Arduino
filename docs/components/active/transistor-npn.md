# NPN 트랜지스터 (2N2222, BC547, S8050)

## 개요

NPN 트랜지스터는 N형-P형-N형 반도체가 샌드위치 구조로 이루어진 양극성 접합 트랜지스터(BJT)입니다.  
베이스(Base) 단자의 소전류로 컬렉터(Collector)와 이미터(Emitter) 사이의 대전류를 제어하는 전류 증폭 소자입니다.  
아두이노에서는 3.3V GPIO 신호로 5V 이상의 부하(LED, 릴레이, 모터, 부저)를 구동하는 스위치로 주로 활용합니다.

**NPN 동작**: 베이스에 전류(IB)가 흐르면 컬렉터에서 이미터로 큰 전류(IC)가 흐릅니다.  
**스위칭 모드**: 포화(Saturation) 영역에서 ON/OFF 스위치로 동작.  
**증폭 모드**: 활성(Active) 영역에서 선형 증폭기로 동작.

## 핀 구성

### TO-92 패키지 (가장 일반적)

```
평평한 면을 앞으로, 핀이 아래를 향하게 잡았을 때:

  ┌─────────┐
  │  평평면 │
  └────┬────┘
    1  2  3  (핀 번호)

모델별 핀 배열:
  2N2222A : 1=E(이미터), 2=B(베이스), 3=C(컬렉터)
  BC547   : 1=C(컬렉터), 2=B(베이스), 3=E(이미터)
  S8050   : 1=E(이미터), 2=B(베이스), 3=C(컬렉터)
  2N3904  : 1=E(이미터), 2=B(베이스), 3=C(컬렉터)
```

> **모델마다 핀 배열이 다릅니다.** 반드시 데이터시트 확인 필수!

| 단자 | 기호 | 기능 |
|------|------|------|
| 베이스 | B | 제어 입력. 소전류(IB)로 트랜지스터 ON/OFF 결정 |
| 컬렉터 | C | 전류 출력. 부하(모터, LED, 릴레이)가 연결됨 |
| 이미터 | E | 전류 출력. 항상 GND 또는 가장 낮은 전위에 연결 |

## 동작 원리

### 세 가지 동작 영역

| 영역 | 베이스 조건 | VCE | 상태 | 용도 |
|------|-----------|-----|------|------|
| 차단 (Cutoff) | IB = 0 (VBE < 0.6V) | = VCC | 완전 OFF | 스위치 OFF |
| 포화 (Saturation) | IB 충분 (IC < hFE × IB) | 0.1~0.3V | 완전 ON | 스위치 ON |
| 활성 (Active) | IB 작음 | VCC > VCE > VCE(sat) | 선형 증폭 | 아날로그 증폭 |

### 스위칭 동작 원리

```
VBE 임계값: 0.6V (실리콘 기준)
VBE < 0.6V → 트랜지스터 OFF (차단)
VBE ≥ 0.6V → 트랜지스터 ON (포화 방향)

전류 증폭 관계:
  IC = hFE × IB   (활성 영역)
  IC(max) ≤ hFE × IB (포화 영역에서는 성립 안 함)
```

## 아두이노 연결 방법

### 베이스 저항 계산

스위칭 회로에서 베이스 저항(RB)을 계산하는 방법:

```
1. 부하 전류 파악: IL (LED라면 10~20mA, 릴레이라면 50~100mA 등)

2. 포화 조건: IB ≥ IC / hFE
   → 실제로는 2~10배 과포화 (β 변동 및 온도 보상)
   → IB(실제) = IL / (hFE/10) ~ IL / (hFE/3)

3. 베이스 저항:
   RB = (VIO - VBE) / IB
   VIO  : GPIO 출력 전압 (3.3V)
   VBE  : 0.6V (실리콘 트랜지스터 고정)

예시: IL = 100mA, hFE = 100, VIO = 3.3V
   IB(필요) = 100mA / 100 = 1mA
   IB(과포화) = 5mA (5배 여유)
   RB = (3.3 - 0.6) / 0.005 = 540Ω → 470Ω 또는 560Ω
```

### 기본 스위칭 회로 (NPN, Active HIGH)

```
부하 (모터/LED/릴레이)
    │
   [VCC]
    │
   [부하]
    │
    ├── 컬렉터 (C)
    │   NPN 트랜지스터
    ├── 이미터 (E) ──── GND
    │
GPIO ──[RB=470Ω]── 베이스 (B)
```

- GPIO HIGH (3.3V) → VBE = 0.6~0.7V → 트랜지스터 ON → 부하에 전류 흐름
- GPIO LOW (0V) → VBE ≈ 0V → 트랜지스터 OFF → 부하 전류 차단

### 유도성 부하 (모터, 릴레이) 역기전력 보호

```
[VCC]
  │
[릴레이 코일]── 플라이백 다이오드(1N4007) ──┐
  │                                         │
컬렉터 (C)                              [VCC]
  NPN 트랜지스터
이미터 (E) ──── GND
```

- 유도성 부하는 전원 차단 시 역기전력(스파이크)을 발생시킵니다.
- 플라이백 다이오드(1N4007)를 부하와 병렬로 역방향 연결하여 보호합니다.

## 코드 예제 개념

### LED 스위칭

```cpp
const int TRAN_PIN = 7;    // 베이스 제어 핀

void setup() {
    pinMode(TRAN_PIN, OUTPUT);
    digitalWrite(TRAN_PIN, LOW); // 초기 OFF
}

void loop() {
    digitalWrite(TRAN_PIN, HIGH); // 트랜지스터 ON → LED ON
    delay(1000);
    digitalWrite(TRAN_PIN, LOW);  // 트랜지스터 OFF → LED OFF
    delay(1000);
}
```

### PWM으로 모터 속도 제어

```cpp
const int MOTOR_PIN = 5;
const int PWM_CHANNEL = 0;
const int PWM_FREQ = 1000;   // 1kHz
const int PWM_BITS = 8;

void setup() {
    ledcSetup(PWM_CHANNEL, PWM_FREQ, PWM_BITS);
    ledcAttachPin(MOTOR_PIN, PWM_CHANNEL);
}

void setMotorSpeed(int speed) {  // 0~255
    ledcWrite(PWM_CHANNEL, speed);
}

void loop() {
    setMotorSpeed(128);  // 50% 속도
    delay(2000);
    setMotorSpeed(255);  // 100% 속도
    delay(2000);
    setMotorSpeed(0);    // 정지
    delay(2000);
}
```

| 멀티미터 다이오드 모드 | B→C, B→E 방향으로 0.6~0.7V 측정되면 NPN |
| hFE 측정 | 멀티미터 hFE 소켓에 꽂아 전류 이득 확인 |

## 주의사항

1. **핀 배열 필수 확인**: 모델마다 핀 순서(EBC 또는 CBE)가 다릅니다. 반드시 데이터시트 참조.
2. **베이스 저항 필수**: 저항 없이 GPIO 직결 시 베이스에 과전류 → 트랜지스터 및 GPIO 핀 손상.
3. **VCE(sat) 고려**: 포화 시 VCE = 0.2~0.5V. 컬렉터 쪽 전원 전압이 이 값보다 충분히 높아야 함.
4. **최대 전류 준수**: IC(max) 초과 시 즉시 손상. 안전 마진 70% 이하로 사용 권장.
5. **열 발생**: IC가 높을수록 발열이 심합니다. Ptot = VCE × IC. 500mW 이상이면 방열판 필요.
6. **플라이백 다이오드**: 릴레이, 모터, 솔레노이드 등 유도성 부하에는 반드시 역기전력 보호 다이오드 추가.
7. **MOSFET과 비교**: 고주파 스위칭이나 완전한 0V 스위칭(VCE=0)이 필요하면 MOSFET이 더 적합.

## 자주 쓰이는 회로 패턴

### 패턴 1: LED 드라이버 (3.3V GPIO → 5V LED)

```
5V ──[330Ω]── LED+ ── LED− ── C(컬렉터)
                              NPN
                              E(이미터) ── GND
GPIO3.3V ──[1kΩ]── B(베이스)
```

### 패턴 2: 릴레이 드라이버

```
5V ──[릴레이 코일]──[1N4007(역방향)]── C(컬렉터)
                                      NPN
                                      E(이미터) ── GND
GPIO ──[1kΩ]── B(베이스)
```

### 패턴 3: 부저 구동

```
5V ──[부저]── C(컬렉터)
              NPN
              E(이미터) ── GND
GPIO ──[1kΩ]── B(베이스)
```

### 패턴 4: 달링턴 배열 (초고이득, hFE1 × hFE2)

```
GPIO ──[RB]── B1(NPN1)
              C1 ── B2(NPN2) ── GND
                    C2 ──[부하]── VCC
                    E2 ── GND
```
- 두 NPN을 직렬로 연결. 전류 이득 = hFE1 × hFE2 (수만 배).
- 소전류 GPIO로 대전류 부하 직접 제어 가능.
- 통합 IC: ULN2003, ULN2803 (달링턴 어레이).
