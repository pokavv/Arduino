# Arduino IDE 설치 및 기본 설정

## Arduino IDE 설치

1. [https://www.arduino.cc/en/software](https://www.arduino.cc/en/software) 에서 다운로드
2. 설치 후 실행
3. 보드를 USB로 연결
4. **도구 → 보드** 에서 사용하는 보드 선택
5. **도구 → 포트** 에서 COM 포트 선택

---

## 화면 구성

```
[체크 버튼]  컴파일만 (업로드 안 함)
[→ 버튼]    컴파일 + 업로드
[시리얼 모니터 버튼]  우측 상단
```

---

## 첫 업로드 테스트

**파일 → 예제 → 01.Basics → Blink** 열고 업로드
→ 내장 LED가 깜빡이면 성공

---

## 시리얼 모니터

**도구 → 시리얼 모니터** (단축키: `Ctrl + Shift + M`)

코드와 시리얼 모니터의 통신 속도가 **반드시 일치**해야 합니다.

```cpp
Serial.begin(115200);  // 코드의 속도
// 시리얼 모니터 우하단에서도 115200 선택
```
